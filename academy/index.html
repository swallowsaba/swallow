§02

推奨パスウェイ Recommended Pathway

順序は絶対ではありませんが、リテラシー → 実装 → データサイエンス → クラウド/エッジ の流れが堅実です。並行学習も可能。

PHASE 01
リテラシー
G検定

AI 用語・歴史・倫理・法律を一気に俯瞰。事業文脈で語れるようになる。

PHASE 02
実装理論
E資格

数学 → ML → DL基礎/応用 → 運用環境。PyTorch / TF で書ける深さを獲得。

PHASE 03
データ基盤
統計検定 2級 → 準1級

推測統計・回帰・多変量。E資格の数学的基礎を統計側から補強。

PHASE 04
Pythonの底力
Python 3 認定 基礎 → 実践

言語仕様、データ分析、実装パターンを段階的に厳密化。

PHASE 05
クラウドML
AWS MLA-C01

SageMaker でのデプロイ・MLOps・モニタリング・セキュリティを把握。

PHASE 06
専門の翼
画像処理検定 / Jetson AI

CV／エッジAI の専門性で差別化。Kaggle ／ ハンズオンと相互補強。

§03

5つの学習フレーム Five Frames

基本 — Basics

各資格に共通する数理・概念・用語。線形代数、確率統計、最適化、機械学習の枠組み、深層学習の構成要素を最短で押さえる。

基本へ

応用 — Advanced

Transformer、拡散モデル、強化学習、RAG、Agent、MLOps、責任あるAI。LLM 時代の中核と実務上の論点を扱う。

応用へ

シミュレーション — Simulation

勾配降下・活性化関数・過学習・決定境界・バックプロパゲーション・Attention・K-means・PCA — 8つのブラウザ内インタラクティブで、数式を「動かして」理解する。

シミュレーションへ

ハンズオン — Hands-on

Colab / PyTorch / SageMaker / Jetson での実装手順。コードを写経し、改造し、自分の問題に適用する。

ハンズオンへ
