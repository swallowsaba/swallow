シミュレーションMove the dial. Watch the model change.

学習率を上げると何が壊れるのか。活性化関数を変えるとどう変わるのか。誤差はどう逆伝播し、Attentionはどこを見るのか。パラメータを動かすことでしか見えない挙動を、8 つの対話的なデモで掴む。

§01 / GRADIENT DESCENT勾配降下

一次元の凸／非凸関数の上を、勾配降下法がどう降りていくかを観察する。学習率を変えると、収束の速さ・振動・発散の境目が見える。

x=-4.000 loss=16.000

観察のヒント

  • η を小さくする → ゆっくり確実に収束
  • η を大きくする → 振動が大きくなり、ある閾値で発散
  • 非凸関数では初期位置によって到達する谷が変わる（局所最適）

§02 / ACTIVATIONS活性化関数の比較

Sigmoid、Tanh、ReLU、Leaky ReLU、GELU、Swish の出力と微分を重ねて表示。それぞれの勾配消失のしやすさを目視で比較する。

勾配消失の正体

  • sigmoid・tanh は飽和域で微分がほぼ 0 → 深い層では勾配が指数的に小さくなる
  • ReLU は正の領域で微分 1 → 勾配消失しにくいが、負の領域は完全に死ぬ（dying ReLU）
  • Leaky ReLU・GELU・Swish はその欠点を緩和した派生

§03 / OVERFIT過学習の可視化

正弦波 + ノイズに対して、多項式の次数を上げていくと何が起きるか。低次は表現力不足（バイアス高）、高次は学習データに過剰適合（バリアンス高）。

train=- test=-

バイアス・バリアンスのトレードオフ

  • 次数を上げる → train 誤差は下がるが、test 誤差は U 字に
  • データ数を増やす → 高次でも過学習しにくくなる
  • ノイズを増やす → 低次の方が頑健に

§04 / DECISION BOUNDARY決定境界

2クラス分類で、モデルの容量（隠れ層のユニット数）を変えると決定境界がどう変わるか。簡易MLP（1隠れ層）で、ランダム重みからの学習を可視化。

epoch=0 loss=-

容量と表現力

  • ユニット数が少ない → 単純すぎて非線形境界を描けない
  • ユニット数を増やす → 複雑な境界も描けるが、過学習リスクも上昇
  • 同じ構造でも初期値次第で違う解に収束（最適化の非凸性）

§05 / BACKPROPバックプロパゲーション可視化

2層 MLP の順伝播と逆伝播を、重みエッジの色と太さに反映して観察する。順方向は活性化値、逆方向は勾配ノルムが流れる様子を可視化する。

ŷ=- loss=-

読み取り方

  • 順伝播モード：エッジの太さ＝活性化×重み の絶対値。出力までの情報の流れが見える
  • 逆伝播モード：エッジの太さ＝勾配の絶対値。誤差が出力側から入力側へ伝播する
  • 1ステップ学習で重みが更新され、損失が下がる過程を観察できる

§06 / ATTENTIONSelf-Attention重み可視化

系列長 N のトークン列に対し、各クエリ位置 i が他のどの位置 j に重みを置くかをヒートマップで表示。スケール (√d_k) と softmax の効果が直感的に分かる。

N=8 τ=1.00

読み取り方

  • 各行＝クエリ位置 i、各列＝キー位置 j。行の総和は1（softmaxの性質）
  • 温度を下げる → 分布が鋭くなり、特定位置に集中。上げる → 一様に近づく
  • 因果マスクは上三角が0（自己回帰生成で未来を見ない）

§07 / KMEANSK-meansクラスタリング

2次元データに対し、重心の初期化→割当→重心更新の反復を1ステップずつ進める。K の選択や初期化次第で結果が大きく変わることを体験する。

iter=0 inertia=-

K-means の弱点

  • 等方的・等サイズ・凸クラスタを仮定。同心円や三日月では失敗しやすい
  • 初期化次第で局所解に落ちる → K-means++ や複数回試行が実務的
  • K は事前指定。エルボー法／シルエット係数／Gap 統計量で評価

§08 / PCA主成分分析の幾何

2次元データの分布から主成分（共分散行列の固有ベクトル）を計算し、第一主成分／第二主成分を視覚化する。回転・スケール・形状でどう変化するか体感する。

λ₁=- λ₂=- 説明率=-%

固有値の意味

  • λ₁（大きい方）＝第一主成分方向の分散。射影が最も広がる方向
  • λ₁ / (λ₁+λ₂) ＝ 第一主成分の累積寄与率（説明率）
  • 非線形構造（曲線・三日月）には PCA は不適 → カーネルPCA・t-SNE・UMAP が必要
