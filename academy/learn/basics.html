ML の本は無数にありますが、骨格は驚くほどコンパクトです。数学 →（モデル化）→ 最適化 →（評価）→ 改善。この5つの動詞のあいだを行き来できれば、新しい技術もすぐに位置づけられます。

§01 / LINEAR ALGEBRA線形代数

ML の大半は線形代数の操作で書けます。ベクトルは「特徴量の束」、行列は「線形変換」、固有値分解と特異値分解は「分解して見る」装置。

  • ベクトル空間：和とスカラ倍が閉じている世界。特徴量空間そのもの。
  • ノルム：L1（曼哈頓距離）、L2（ユークリッド距離）、L∞（最大値）。正則化や距離計算で頻出。
  • 内積とコサイン類似度：埋め込みの近さを測る基本道具。
  • 固有値・固有ベクトル：「変換しても向きが変わらない方向」と「その伸縮率」。共分散行列に適用すれば PCA に。
  • SVD：任意の行列を直交変換と対角化に分解。低ランク近似、画像圧縮、推薦の数理。

手で計算するのは 2×2、3×3 で十分。「式の意味」を絵で説明できるかが大事。

§02 / PROBABILITY確率と統計

  • 条件付き確率とベイズの定理：データから仮説を更新する文法。RAG・分類・異常検知の根底。
  • 代表的な分布：ベルヌーイ（コイン）、二項（コインの和）、ポアソン（稀なイベント）、正規（自然界の合計）、指数（待ち時間）。
  • 期待値・分散・共分散：データの「重心」と「広がり」と「連動」。
  • 大数の法則と中心極限定理：標本平均は母平均に近づき、その分布は正規に近づく。検定・区間推定の根拠。
  • 推定論：最尤推定（MLE）、最大事後確率推定（MAP）、ベイズ推定。損失関数の正体は対数尤度であることが多い。

§03 / INFORMATION情報理論

  • 情報量：起こりにくい事象ほど情報量が大きい（−log p）。
  • エントロピー H(X)：分布の不確かさ。一様分布で最大、ピーキーな分布で小さい。
  • クロスエントロピー H(P, Q)：真の分布 P を Q で符号化したときの平均符号長。分類の損失関数そのもの。
  • KL ダイバージェンス D(P||Q)：P と Q の「距離」のような量（非対称）。VAE、強化学習、知識蒸留で頻出。

§04 / OPTIMIZATION最適化

ML の学習は「損失関数を最小化する」最適化問題。勾配が指す方向に少しずつ下るのが基本。

  • 勾配降下法：θ ← θ − η ∇L(θ)。学習率 η が大きすぎれば発散、小さすぎれば収束遅延。
  • 確率的勾配降下（SGD）：ミニバッチで近似勾配を計算。並列化・正則化的効果。
  • Momentum / Adam：過去の勾配を活かして加速。Adam は学習率まで適応化。
  • 凸性：凸関数なら局所最適＝大域最適。DL の損失は非凸だが、SGD でうまくいく経験則がある。

§05 / FRAMEWORK機械学習の枠組み

どんなアルゴリズムも、3つの問いに答えています。

  1. 仮説集合（モデル）：どんな関数を候補にするか
  2. 損失（指標）：どの候補が「よい」かをどう測るか
  3. 探索（学習）：候補空間をどう探すか

例：線形回帰 = 一次関数の集合 + 二乗誤差 + 正規方程式 or 勾配降下。SVM = 超平面 + ヒンジ損失 + 双対最適化。ニューラルネット = 合成関数 + クロスエントロピー + 逆伝播 + Adam。新しい技術もこの枠で読むと整理できます。

3 つの学習パラダイム

  • 教師あり：入力 x と正解 y のペアから写像 f を学ぶ。分類・回帰。
  • 教師なし：x だけから構造を見出す。クラスタリング・次元削減・密度推定。
  • 強化学習：環境との相互作用から、累積報酬を最大化する方策を学ぶ。

近年は自己教師あり学習（self-supervised）が中心になっており、データ自身からラベルを作って学ぶことで、巨大コーパスを活用しています。BERT・GPT・SimCLR・MAE はみなこの枠組み。

§06 / DL ELEMENTS深層学習の構成要素

DL モデルは結局、以下の積み木の組み合わせ。

  • 線形変換：y = Wx + b。「特徴量の混ぜ合わせ」
  • 非線形活性化：ReLU / GELU / Softmax。「曲げる」
  • 正規化：BN / LN / GN。「スケールを揃える」
  • 畳み込み：局所的なパターン抽出。並進不変性。
  • アテンション：トークン同士の関係を学習。位置を超えた依存。
  • 残差接続：恒等写像をベースラインに置き、勾配を通す。
  • 埋め込み（embedding）：離散シンボルを連続空間に写像する。

「Transformer」「ViT」「Diffusion」もすべて、これらを組み合わせて作られています。

§07 / METRICS評価指標

分類

  • Accuracy — クラスバランスがよい時の基本
  • Precision・Recall・F1 — 不均衡データ、誤検知／見落としに敏感な場合
  • ROC-AUC、PR-AUC — 閾値非依存の比較

回帰

  • MAE — 外れ値に頑健
  • MSE / RMSE — 大きな誤差にペナルティ
  • R² — モデルがどれだけ分散を説明したか

生成・順位付け

  • BLEU、ROUGE、METEOR、BERTScore — 自然言語生成
  • FID、IS — 画像生成の品質
  • nDCG、MRR、MAP — 検索・推薦

§08 / REGULARIZATION正則化と汎化

学習データに対して誤差を下げるのは簡単。難しいのは見ていないデータでも誤差が下がること（汎化）。そのための工夫が正則化です。

  • L1 / L2 ペナルティ — 重みを小さく保つ／スパースにする
  • Dropout — ランダムにユニットを消して、共依存を減らす
  • Data Augmentation — 入力に多様性を持たせる
  • Early Stopping — 検証損失が悪化する前で止める
  • Label Smoothing — 過信を抑える
  • Mixup / CutMix — 入力どうしを混ぜる

「過学習が起きている」と感じたら、正則化を強める・データを増やす・モデルを小さくするの3手をまず試す。
