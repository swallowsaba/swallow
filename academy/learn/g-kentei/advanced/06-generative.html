G-KENTEI · ADVANCED · CHAPTER 18

分布を学び、
新しいデータを生む。

生成モデルの本質は『データ分布 p(x) を学習し、そこからサンプリングする』こと。GANは識別器との敵対的学習、VAEは変分推論、Diffusionは逐次的なノイズ除去 ── アプローチは異なるが目的は同じ。本章でこの3つを横断的に理解し、現代の画像・動画・音声生成の仕組みを解読します。

SECTION ONE

基本

生成モデル、GAN、VAE、Diffusion、Flow、Energy-based、自己回帰、画像生成、動画、音声の10節。

基本

§1.1 生成モデルの定義

『データ分布 p_data(x) を学習し、そこからサンプリングする』モデル。識別モデル（p(y|x) を学習）と対照的。代表アプローチ：

  • 陰的分布（GAN）：密度を計算せず、サンプリングのみ
  • 陽的分布（VAE、Flow、Autoregressive）：尤度を計算可能
  • Score-based（Diffusion）：分布の勾配 ∇log p(x) を学習

§1.2 GAN（Generative Adversarial Network）

Goodfellow et al. 2014
　 Generator G(z) → 偽画像
　 Discriminator D(x) → 真偽判定

min-max ゲーム　 min_G max_D 𝔼[log D(x)] + 𝔼[log(1 - D(G(z)))]

理論的均衡で G が真の分布を再現。

主な派生：DCGAN（CNN化）、WGAN（Wasserstein距離）、StyleGAN（高解像度顔生成）、BigGAN（ImageNet）。問題：モード崩壊（多様性消失）、不安定な学習。

§1.3 VAE（Variational Autoencoder）

Kingma & Welling 2013
　 Encoder q(z|x) → 潜在変数 z
　 Decoder p(x|z) → 再構成

ELBO 最大化　 E_q[log p(x|z)] - KL(q(z|x) || p(z))
　　　　　　　 ↑ 再構成損失　 　 ↑ 正則化（事前 N(0,I)）

連続的な潜在空間、滑らかな補間が可能。GANより生成品質はぼやけるが、安定学習＋確率密度計算可能。VQ-VAEは離散化版。

§1.4 Diffusion Model — 現代の主役

Sohl-Dickstein 2015、Ho et al. 2020（DDPM）、Song et al. 2021（Score-based SDE）。

順方向過程（学習）：x₀ にガウシアンノイズを T 回加えて x_T（純粋ノイズ）
　 x_t = √(α_t) x_{t-1} + √(1-α_t) ε

逆方向過程（生成）：NN がノイズを逐次予測し、ノイズから x₀ を復元
　 学習：NN ε_θ(x_t, t) が x_t から ε を予測
　 損失：E[||ε - ε_θ(x_t, t)||²]

DDPM、DDIM、Latent Diffusion (Stable Diffusion) などの派生
FIG 18.1 ／ Diffusion Model。順方向で画像→ノイズ（学習用データ）、逆方向でNNがノイズ予測し画像復元（生成）。Stable Diffusion / DALL-E 2 / Imagen の核。

§1.5 Flow-based Models

Normalizing Flow（NICE, RealNVP, Glow）：可逆変換 f: z → x を学習。厳密な尤度計算が可能。条件付き生成・密度推定で重要だが、画像生成の品質は Diffusion に劣る。

§1.6 Energy-based Models（EBM）

p(x) ∝ exp(-E(x)) でエネルギー関数 E を学習。Langevin Dynamics 等でサンプリング。Score-based Diffusion の親戚。Boltzmann Machine の現代版。

§1.7 自己回帰モデル

p(x) = Π p(xᵢ | x_{<i}) で分解、トークンを順次予測。PixelRNN, PixelCNN（画像）、WaveNet（音声）、GPT（テキスト）。生成は遅いが厳密尤度可能。

SECTION TWO

応用

Latent Diffusion、ControlNet、LoRA、評価指標の4節。

応用

§2.1 Latent Diffusion — Stable Diffusion の核心

Rombach et al. 2022。VAE で画像を低次元潜在空間（典型 64×64×4）に圧縮し、潜在空間で Diffusion。計算量を1/64に削減しつつ品質維持。これがコンシューマGPUでの画像生成を可能にした。

§2.2 ControlNet

Zhang et al. 2023。Stable Diffusion に 姿勢・エッジ・深度などの追加制約を与える条件付け技術。OpenPose の骨格→人物画像、線画→塗り絵、深度マップ→3D 感のある画像など、生成の制御性を劇的に向上。

§2.3 LoRA for Diffusion

LLM の LoRA をDiffusionに適用。10〜100枚の画像で特定キャラ・スタイルを学習、メモリ効率良く配布可能（数MB）。Civitai 等で大量公開。

§2.4 生成画像の評価

  • FID（Fréchet Inception Distance）：Inception 特徴量の分布距離、画像品質と多様性
  • IS（Inception Score）：品質と多様性のバランス（古い）
  • CLIP Score：テキスト-画像整合性
  • 人間評価：A/B 比較、最終的に最も信頼
  • PickScore, ImageReward：人間嗜好の学習モデル

ハンズオン

§3.2 GAN スクラッチ

import torch import torch.nn as nn class Generator(nn.Module): def __init__(self, z_dim=100, img_dim=784): super().__init__() self.net = nn.Sequential( nn.Linear(z_dim, 256), nn.LeakyReLU(0.2), nn.Linear(256, 512), nn.LeakyReLU(0.2), nn.Linear(512, img_dim), nn.Tanh()) def forward(self, z): return self.net(z) class Discriminator(nn.Module): def __init__(self, img_dim=784): super().__init__() self.net = nn.Sequential( nn.Linear(img_dim, 512), nn.LeakyReLU(0.2), nn.Linear(512, 256), nn.LeakyReLU(0.2), nn.Linear(256, 1), nn.Sigmoid()) def forward(self, x): return self.net(x) # min-max ゲーム学習 G, D = Generator(), Discriminator() opt_g = torch.optim.Adam(G.parameters(), lr=2e-4) opt_d = torch.optim.Adam(D.parameters(), lr=2e-4) bce = nn.BCELoss()

§3.3 VAE 実装

class VAE(nn.Module): def __init__(self, x_dim=784, z_dim=20): super().__init__() self.encoder = nn.Sequential(nn.Linear(x_dim, 400), nn.ReLU()) self.fc_mu = nn.Linear(400, z_dim) self.fc_logvar = nn.Linear(400, z_dim) self.decoder = nn.Sequential(nn.Linear(z_dim, 400), nn.ReLU(), nn.Linear(400, x_dim), nn.Sigmoid()) def reparameterize(self, mu, logvar): std = torch.exp(0.5 * logvar) eps = torch.randn_like(std) return mu + eps * std def forward(self, x): h = self.encoder(x) mu, logvar = self.fc_mu(h), self.fc_logvar(h) z = self.reparameterize(mu, logvar) return self.decoder(z), mu, logvar def vae_loss(recon, x, mu, logvar): bce = nn.functional.binary_cross_entropy(recon, x, reduction='sum') kld = -0.5 * torch.sum(1 + logvar - mu.pow(2) - logvar.exp()) return bce + kld

SECTION FOUR

シミュレーション

GANモード崩壊、Diffusionステップ、CLIP-guidedの3節。

シミュレーション

SIM 1 — GAN のモード崩壊

>> MNIST GAN を素朴に学習

[初期] 全クラスのノイズを生成
[1000 step] 数字 3,5,8 のみ生成（モード崩壊開始）
[5000 step] 数字 8 ばかり生成（厳しいモード崩壊）

>> G が D を騙しやすいモードに局所最適
>> 他クラスは生成不能、多様性消失

[対策] WGAN-GP, Mini-batch Discrimination, Spectral Norm
　 改善後 → 10クラス均等に生成可能

SIM 2 — Diffusion のサンプリングステップ

>> Stable Diffusion で同プロンプト、ステップ数を変える

ステップ 10:　 ノイジー、構図確定せず
ステップ 20:　 構図見える、細部不安定
ステップ 30:　 実用品質
ステップ 50:　 高品質、デフォルト
ステップ 100: わずかに改善、計算コスト2倍
ステップ 250: 通常 50 と差なし

>> DDIM, DPM-Solver などで 10-20 ステップでも高品質に
>> Distillation で 1-4 ステップ生成も実用化（SDXL Turbo）

SIM 3 — Classifier-Free Guidance の効果

>> Stable Diffusion、guidance_scale を変化

scale 1.0: プロンプトをあまり聞かない、自由な生成
scale 7.5: バランス（デフォルト）
scale 15:　プロンプトに忠実、品質やや低下
scale 30:　過剰、不自然なアーティファクト

>> CFG は条件付き/無条件の差を増幅
>> 7-10 が typical、強いプロンプトに合わせて調整
SECTION FIVE

問題

G検定形式 20問。

問題

PROBLEM 1
GAN の基本構造は。
① エンコーダのみ　 ② Generator と Discriminator の敵対的学習　 ③ デコーダのみ　 ④ クラスタリング
ANSWER ② Goodfellow 2014。Generator が偽画像、Discriminator が真偽判定、min-max ゲーム。
PROBLEM 2
GAN の典型的な問題で『多様性が失われる』現象を何と呼ぶか。
① 過学習　 ② モード崩壊（Mode Collapse）　 ③ 勾配消失　 ④ 不確実性
ANSWER ② Generator が騙しやすい一部モードに局所最適。WGAN-GP、Spectral Normalization、Mini-batch Discrimination で対策。
PROBLEM 3
VAE の損失関数 ELBO は何と何の組合せか。
① 再構成損失 + KL ダイバージェンス（事前との距離）　 ② MSE のみ　 ③ Hinge　 ④ 敵対的損失
ANSWER ① ELBO = 再構成 - KL(q(z|x)||p(z))。再構成精度と潜在分布の事前 N(0,I) への近さのバランス。
PROBLEM 4
Diffusion Model の学習は何を予測するか。
① ラベル　 ② 画像に加えられたノイズ（または score = ∇log p(x)）　 ③ クラスタ　 ④ Bounding Box
ANSWER ② 各タイムステップで NN がノイズを予測し、それを差し引いて x_{t-1} を得る。これを T 回繰り返してノイズから画像復元。
PROBLEM 5
Stable Diffusion の核心技術は。
① ピクセル空間のDiffusion　 ② Latent Diffusion（VAEで圧縮した潜在空間で Diffusion）　 ③ GAN　 ④ Flow
ANSWER ② Rombach 2022。512×512 を 64×64 潜在に圧縮し、計算量1/64に。コンシューマGPUで動作可能に。
PROBLEM 6
ControlNet の役割は。
① 高速化　 ② Stable Diffusion に追加制約（姿勢、深度、エッジ）を与えて生成を制御　 ③ パラメータ削減　 ④ 学習率自動
ANSWER ② Zhang 2023。骨格→人物、線画→塗り絵、深度→立体感のある画像など。生成の制御性を劇的向上。
PROBLEM 7
『Classifier-Free Guidance（CFG）』の役割は。
① 分類器学習　 ② 条件付き/無条件のスコア差を増幅、テキスト指示への忠実度を高める　 ③ クラスタ　 ④ 学習率
ANSWER ② Ho 2022。guidance_scale パラメータで強さを調整。7-10 が典型、高すぎると過剰でアーティファクト発生。
PROBLEM 8
DALL-E 2 の主な構成要素は。
① CNN のみ　 ② CLIP（テキスト・画像埋め込み）＋ Diffusion （prior + decoder）　 ③ RNN のみ　 ④ K-means
ANSWER ② CLIP埋め込みを介してテキストと画像を結ぶ、unCLIP アーキテクチャ。
PROBLEM 9
画像生成評価の代表的指標『FID』は何を測るか。
① 学習速度　 ② Inception 特徴量で実画像分布と生成画像分布の距離　 ③ クラス数　 ④ パラメータ数
ANSWER ② Fréchet Inception Distance。画像品質と多様性を一値に集約。低いほど良い。
PROBLEM 10
『CLIP Score』が評価するのは。
① 画像品質のみ　 ② テキストプロンプトと生成画像の意味的整合性　 ③ 計算速度　 ④ ファイルサイズ
ANSWER ② CLIP モデルの埋め込み類似度。プロンプト忠実度の標準指標。
PROBLEM 11
Sora や Veo は何のためのモデルか。
① 音楽生成　 ② 動画生成（テキスト→1分以上の高品質動画）　 ③ 顔認識　 ④ 文章生成
ANSWER ② OpenAI Sora（2024）、Google Veo。Diffusion Transformer ベース、物理理解・長時間生成が特徴。
PROBLEM 12
Whisper（OpenAI）の用途は。
① 画像生成　 ② 多言語音声認識（STT）　 ③ TTS　 ④ 翻訳のみ
ANSWER ② Speech-to-Text の多言語モデル。99言語対応、ノイズに頑健。オープンウェイト。
PROBLEM 13
VQ-VAE の特徴は。
① 連続潜在　 ② 離散コードブックで潜在表現、画像・音声生成や Tokenizer で活用　 ③ 線形変換　 ④ ノイズなし
ANSWER ② 連続潜在を離散コードに量子化。Audio LM、画像 Tokenizer などで使用。
PROBLEM 14
Normalizing Flow の利点は。
① 厳密な尤度計算が可能（可逆変換）　 ② 高速生成　 ③ パラメータ少　 ④ 学習不要
ANSWER ① 可逆な変換 f: z → x を学習し、変数変換公式で厳密尤度。GANやVAEと違い密度評価可能。画像品質は Diffusion に劣る。
PROBLEM 15
Diffusion Models の高速化『Distillation』が達成するのは。
① 学習速度のみ　 ② 50ステップを 1-4 ステップに圧縮、リアルタイム生成可能に　 ③ パラメータ削減　 ④ 不可能
ANSWER ② 教師モデルから生徒モデルへの蒸留。SDXL Turbo、LCM などでリアルタイム画像生成（30fps）が可能に。
PROBLEM 16
LoRA を Diffusion に適用する利点は。
① 全パラメータ学習　 ② 10-100枚の画像で特定キャラ/スタイル学習、数MBで配布可能　 ③ 高速学習不可　 ④ 精度低下
ANSWER ② Civitai 等で大量のLoRA公開、コミュニティが活発。Stable Diffusion カスタマイズの主流。
PROBLEM 17
StyleGAN の特徴は。
① 全結合のみ　 ② スタイル（属性）と画像内容を分離、高解像度顔生成、潜在空間編集　 ③ 線形　 ④ RNN
ANSWER ② NVIDIA、Karras 2019。ThisPersonDoesNotExist のあの顔。潜在空間の意味的編集（笑顔強度、年齢など）。
PROBLEM 18
音楽生成モデル MusicGen は何で学習されているか。
① 画像のみ　 ② 大量の楽曲とテキスト説明のペア、自己回帰モデル　 ③ ランダム　 ④ MIDI のみ
ANSWER ② Meta、2023。テキスト→音楽の Transformer ベース。Suno、Udio などのサービスも類似アプローチ。
PROBLEM 19
生成 AI の社会的課題として該当しないのは。
① ディープフェイク　 ② 著作権　 ③ バイアス　 ④ 必ず物理法則を守る
ANSWER ④ 生成モデルは物理法則を必ずしも守らない（手指の数間違い、不自然な液体運動など）。Sora は改善したが完璧でない。
PROBLEM 20
Diffusion Model が現在 GAN を凌駕している主な理由は。
① 学習が速い　 ② 学習安定、高品質、多様性確保、テキスト条件付け容易、スケール可能　 ③ パラメータ少　 ④ メモリ少
ANSWER ② GANの不安定学習・モード崩壊の問題なし、Diffusion Transformer でスケール可能（Sora）。現代生成AIの主流。

章のまとめ

生成モデルの3大系譜 GAN、VAE、Diffusion を扱いました。GAN は敵対的学習、VAE は変分推論、Diffusion はノイズ除去。Stable Diffusion の Latent Diffusion、ControlNet、LoRA。Sora/Veo の動画生成、Whisper、MusicGen の音声・音楽生成。ハンズオンで Stable Diffusion、GAN、VAE、ControlNet、LoRA。シミュレーションで GAN モード崩壊、Diffusion ステップ、CFG。次章は強化学習 AlphaGo、DQN、PPO、AlphaZero へ。
