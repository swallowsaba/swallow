§1 / MDPマルコフ決定過程

強化学習の数学的定式化。次の5つ組で定義：

  • S：状態空間
  • A：行動空間
  • P(s' | s, a)：遷移確率
  • R(s, a)：報酬関数
  • γ ∈ [0, 1)：割引率（未来の報酬の現在価値）

目標：累積報酬の期待値を最大化する 方策 π(a | s) を見つける。

J(π) = E_π [Σₜ γᵗ rₜ]

§2 / Q価値関数と Q 学習

状態価値関数 V^π(s) と 行動価値関数 Q^π(s, a)：

V^π(s) = E_π[Σₜ γᵗ rₜ | s₀ = s]
Q^π(s, a) = E_π[Σₜ γᵗ rₜ | s₀ = s, a₀ = a]

Q 学習（Watkins, 1989）：最適 Q* を反復的に学習

Q(s, a) ← Q(s, a) + α [r + γ max_{a'} Q(s', a') - Q(s, a)]

§3 / DQNDQN（Deep Q-Network）

2013年、DeepMind が Q 関数を ディープニューラルネットで近似。Atari ゲームを人間以上のスコアで攻略し、深層強化学習の幕開け。

主要な工夫：

  • 経験再生（Replay Buffer）：過去の経験をランダムサンプリングして学習を安定化
  • ターゲットネットワーク：固定された Q' で TD ターゲット計算
  • Double DQN：行動選択と価値推定を分離して過大評価を緩和

§4 / POLICY方策勾配と PPO

方策 π を直接最適化（価値関数を経由しない）：

∇J(θ) = E_π[∇log π_θ(a | s) · Q(s, a)]

これが REINFORCE の基本式。Q を Advantage A = Q - V に置き換えると分散が下がる。

4.1 PPO（Proximal Policy Optimization）

2017年、OpenAI が提案。方策の更新を KL で制約 することで安定化。

§5 / ACActor-Critic 系

方策（Actor）と価値（Critic）を両方学習する手法：

  • A2C / A3C：同期・非同期 Advantage Actor-Critic
  • SAC（Soft Actor-Critic）：エントロピー最大化で探索を促進
  • DDPG / TD3：連続行動空間向け（ロボット制御）

§6 / ALPHAAlphaGo / AlphaZero / MuZero

AlphaGo (2016)人間の棋譜で事前学習 + 自己対戦 RL。Lee Sedol に勝利
AlphaGo Zero (2017)人間データなし、ゼロからの自己対戦のみで世界最強
AlphaZero (2017)囲碁・将棋・チェスを統一アルゴリズムで攻略
MuZero (2019)環境モデルも学習。Atari もカバー
AlphaStar (2019)StarCraft II でグランドマスター級

共通技術：モンテカルロ木探索（MCTS） + 方策・価値ネット + 自己対戦。

§8 / DPODPO と新世代手法

RLHF は実装が複雑で不安定。これを 「報酬モデル不要、直接最適化」 に簡略化したのが DPO（Direct Preference Optimization, 2023）。

L_DPO = -E[log σ(β log(π(y_w|x)/π_ref(y_w|x)) - β log(π(y_l|x)/π_ref(y_l|x)))]

y_w：好まれた応答、y_l：好まれなかった応答。RLHF と同等以上の性能で、実装が遥かに簡単。

派生：IPO, KTO, ORPO, SimPO など、2024年も活発に研究が続く。

関連：Constitutional AI（Anthropic）

「AI 自身に憲法（原則）を与え、自分でフィードバックを生成する」アプローチ。人間ラベラーの負担を減らしつつ、価値観を明示的に組み込む。Claude のアライメントの中核。
