応用 章4 — BERT と GPTTwo Pretraining Paradigms

2018年、Transformer が 事前学習 + 微調整 という枠組みで爆発的に発展しました。BERT（Google）と GPT（OpenAI）はその二大潮流。それぞれが Transformer のどの部分を使い、どんな自己教師あり学習で力を引き出したかを理解します。

§1 / PARADIGM事前学習＋微調整パラダイム

2018年以前は、各タスクごとに別のモデルを最初から学習していた。2018年以降の標準：

  1. 事前学習：大量のラベルなしテキストで、汎用的な言語表現を学ぶ（自己教師あり）
  2. 微調整（Fine-tuning）：少量のラベル付きデータで、特定タスクに特化

これにより：

  • 少量データでも高性能
  • 1つの基盤モデルから多数のタスクに展開可能
  • 「基盤モデル（Foundation Model）」という概念が成立

§2 / BERTBERT（Bidirectional Encoder）

2018年10月、Google が発表。Bidirectional Encoder Representations from Transformers。Transformer の Encoder のみ を使用。

2.1 事前学習タスク

  • Masked Language Model（MLM）：入力の15%を [MASK] に置き換え、それを予測。双方向の文脈を活用できる
  • Next Sentence Prediction（NSP）：2つの文が連続するかを判定（後に RoBERTa で廃止）
FIG 4.1 BERT のマスク言語モデル。文中の単語を [MASK] に置き換え、両側の文脈から予測する。これにより双方向の文脈を取り込んだ表現が学習される。

2.2 BERT 系の進化

RoBERTa2019, Meta. BERT の学習を改良（大量データ、NSP削除、より長い学習）
ALBERT2019, Google. パラメータ共有で軽量化
DistilBERT2019. 知識蒸留で 40% 軽量化、97% 性能維持
DeBERTa2020, Microsoft. 相対位置エンコーディング
多言語 BERTmBERT, XLM-R など。多言語対応

§3 / GPTGPT（Decoder のみ・自己回帰）

2018年6月、OpenAI が GPT-1 を発表（BERT より少し早い）。Transformer の Decoder のみ。

3.1 事前学習タスク

Next Token Prediction（自己回帰言語モデル）：

P(x₁, x₂, ..., xₙ) = Π P(xₜ | x₁, ..., xₜ₋₁)

つまり「次の単語を予測する」。これだけで膨大な能力が学習される。

3.2 GPT 系の進化と Few-shot 能力

GPT-12018, 117M params. 事前学習＋タスク特化微調整
GPT-22019, 1.5B params. Zero-shot タスク実行を提示
GPT-32020, 175B params. In-context Learning（プロンプトでタスク指示）
InstructGPT / ChatGPT2022. RLHF で人間の好みに調整
GPT-42023, マルチモーダル（画像入力対応）
GPT-4o, o1, o32024-2025, ネイティブマルチモーダル、推論モード

3.3 In-context Learning（ICL）

GPT-3 が示した革命：パラメータを更新せず、プロンプトに例を含めるだけでタスクを実行できる。

  • Zero-shot：「英語を仏訳して: Hello → ?」
  • One-shot：1例だけ提示
  • Few-shot：数例提示

§5 / T5T5・BART：Encoder-Decoder 系統

T5（2019, Google）：すべての NLP タスクを「テキスト → テキスト」に統一。"translate English to German: ..." のようなプロンプト形式。

BART（2019, Meta）：ノイズ入力からの再構成タスクで事前学習。要約・翻訳に強い。

§6 / LINEAGEモデル系統図

Encoder のみBERT (2018) → RoBERTa, ALBERT, DistilBERT, DeBERTa, ELECTRA
Decoder のみGPT-1 (2018) → GPT-2/3/4, LLaMA, Mistral, Claude, Gemini, Qwen, DeepSeek
Encoder-DecoderT5 (2019), BART, mBART, FLAN-T5
ハイブリッドUL2, PaLM 系の一部

2024-2025 の主流は Decoder のみ。ChatGPT・Claude・Gemini・Llama・Mistral すべて。BERT は依然として NLU（理解系）で現役。

§7 / SUMMARY章のまとめ

1事前学習＋微調整：2018年以降の標準パラダイム
2BERT: Encoder のみ、Masked Language Model、双方向
3GPT: Decoder のみ、Next Token Prediction、単方向（自己回帰）
4BERT は理解、GPT は生成に強い
5In-context Learning：GPT-3 が示した、プロンプトだけでのタスク実行
6RoBERTa, ALBERT, DistilBERT, DeBERTa など BERT 派生
7T5/BART は Encoder-Decoder、すべてを text-to-text に
