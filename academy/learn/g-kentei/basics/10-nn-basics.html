§1 / PERCEPTRONパーセプトロン

Frank Rosenblatt（1958年）が提案した最初期のニューラルネット。入力の線形結合にステップ関数を適用：

y = step(wᵀx + b),   step(z) = 1 if z ≥ 0 else 0
FIG 10.1 パーセプトロンの構造。入力 xᵢ に重み wᵢ をかけ、足し合わせてバイアス b を加え、ステップ関数を通す。「重み付き合計が閾値を超えたら発火」というニューロンの数学化。

1.1 学習アルゴリズム

誤分類されたサンプルがあれば重みを更新：

w ← w + η · (y - ŷ) · x

線形分離可能なデータならば 有限ステップで収束することが証明されている（Rosenblatt の収束定理）。

§2 / XORXOR 問題：パーセプトロンの限界

1969年、Minsky と Papert が著書『Perceptrons』で証明：単層パーセプトロンは XOR を学習できない。

FIG 10.2 XOR 問題。出力 1 の点と 0 の点を1本の直線で分離できない（線形分離不可能）。多層化することで解決される。これが第1次AI冬の原因の一つ。

これが第1次AI冬の主要因の一つ。多層化が解決策だが、当時は多層NNの学習法（バックプロパゲーション）が一般化していなかった。

§3 / MLP多層パーセプトロン（MLP）

パーセプトロンを多層に積み、ステップではなく 滑らかな活性化関数（sigmoid/ReLU等）を使うことで、非線形関数を表現できる。

FIG 10.3 2隠れ層の多層パーセプトロン。各層は (重み付き和 + 活性化) の繰り返し。全結合（fully connected, dense）が基本構造。

§4 / UNIV万能近似定理

Cybenko (1989), Hornik (1991)：1隠れ層を持つ MLP は、隠れユニットを十分多く取れば、任意の連続関数を任意の精度で近似できる。

⚠ 「表現できる」と「学習できる」は別
万能近似定理は 「そのような重みの値が存在する」 ことしか保証しない。
実際にそれを 学習で見つけられる かは別問題。
また、1隠れ層で表現可能でも、現実には多層化で遥かに少ないユニット数で表現できる（指数的圧縮）。これが「Deep Learning」と呼ばれる所以。

§6 / DEEPなぜ深くするのか

「1隠れ層で十分」が万能近似定理の主張。それでもなぜ 深層 にするのか？

  • 指数的圧縮：同じ表現力を遥かに少ない総パラメータ数で実現できる
  • 階層的特徴抽出：低層は線分・色、中層は形・テクスチャ、高層は意味的概念 — と抽象度が階層的に上がる
  • 転移学習：事前学習で学んだ低層の特徴は他タスクでも使える
  • 帰納バイアス：層構造そのものが「現実世界の階層性」と合致

§7 / SUMMARY章のまとめ

1パーセプトロン = 線形結合 + ステップ関数。Rosenblatt 1958
2線形分離可能なデータには有限ステップで収束
3単層では XOR を解けない（線形分離不可能）→ Minsky 1969
4多層パーセプトロン（MLP）で非線形を表現
5万能近似定理：1隠れ層で任意の連続関数を近似可能
6ただし「表現可能」と「学習可能」は別
7活性化関数：ReLU が標準、Transformer は GELU、出力は Sigmoid/Softmax
8深層化は「指数的圧縮」と「階層的特徴抽出」で意味がある
