§1 / LINEAR線形回帰

機械学習で最も基本的なモデル。入力 x ∈ ℝᵈ に対し、線形に予測：

ŷ = wᵀx + b

損失：二乗誤差 L = (1/2n) Σ (ŷᵢ - yᵢ)²。正規方程式で閉形式解が存在：

ŵ = (XᵀX)⁻¹ Xᵀy

XᵀX が非正則（多重共線性あり）だと解が不安定 → リッジ回帰で逃げる。

§2 / REGリッジ・ラッソ・Elastic Net

過学習を防ぐため、重みの大きさにペナルティを課す：

Ridge (L²)L + λ||w||² → ŵ = (XᵀX + λI)⁻¹ Xᵀy（解析解あり）
Lasso (L¹)L + λ||w||₁ → 解析解なし、座標降下法。スパース化
Elastic NetL + λ₁||w||₁ + λ₂||w||² → 両者の長所
FIG 8.1 L¹ と L² 正則化の幾何。L¹ の単位球は菱形なので「角」に解が来やすく、多くの係数がゼロ（スパース化）。L² は円なので解は滑らかに分布。

§3 / LOGロジスティック回帰：二値分類

線形出力 z = wᵀx + b を シグモイドで確率に変換：

σ(z) = 1 / (1 + e⁻ᶻ),   P(y=1 | x) = σ(wᵀx + b)

損失は 二値クロスエントロピー（最尤推定と等価）。多クラスは softmax 回帰。

§4 / SVMサポートベクトルマシン

クラス間の マージン（境界からの最近接点までの距離）を最大化する分類器。

FIG 8.2 SVM はクラス境界からの最近接点（サポートベクトル）までの距離を最大化する超平面を見つける。線形分離できない場合は カーネルトリック で高次元に写像。

4.1 カーネルトリック

データを高次元に写像する関数 φ を陽に計算せず、内積 K(x, x') = φ(x)·φ(x') だけを計算 する技法。代表的なカーネル：

  • 線形：K(x, x') = xᵀx'
  • 多項式：K(x, x') = (xᵀx' + 1)ᵈ
  • RBF（ガウシアン）：K(x, x') = exp(-γ ||x - x'||²) — 最も柔軟

§5 / TREE決定木

データを if-then ルールで再帰的に分割。解釈性が高い のが最大の利点。

FIG 8.3 決定木の構造（融資審査の例）。各ノードで1つの特徴量について閾値分割。深い木ほど表現力が増すが過学習しやすい。

分岐基準：

  • 情報利得：分岐前後のエントロピー差（章6）
  • ジニ不純度：1 - Σ pᵢ²。エントロピーと似た性質

§6 / ENSEMBLEアンサンブル：RF と GBDT

6.1 ランダムフォレスト（バギング）

複数の決定木を 並列に学習し、多数決。各木は：

  • ブートストラップ標本（同じデータから復元抽出）で学習
  • 特徴量サブセット（毎ノードで一部だけ考慮）で多様化

結果：単一の木より 過学習しにくく、頑健。実装簡単。

6.2 勾配ブースティング（GBDT）

木を 逐次的に追加し、前の木の誤差（負勾配）を埋めるように学習：

F_m(x) = F_{m-1}(x) + γ_m h_m(x),   h_m は誤差を近似する木

代表実装：

XGBoost2014年。Kaggleの支配的勝者。多くの実務で標準
LightGBM2017年。Microsoft。GOSS + EFB で高速
CatBoost2017年。Yandex。カテゴリ変数の扱いが優秀

表形式データでは いまだに深層学習を凌駕する ことが多い（2024-2025年の Kaggle 上位もGBDT 系が中心）。

§7 / KMEANSK-means クラスタリング

教師なしのクラスタリング。Lloyd アルゴリズム：

  1. K 個のセントロイドをランダム初期化
  2. 割当ステップ：各データを最近接セントロイドに割当
  3. 更新ステップ：各クラスタのセントロイドを平均で更新
  4. 収束まで 2-3 を反復
FIG 8.4 K-means の動作。ランダム初期化したセントロイドを、割当と平均更新で交互に動かすと、データの自然なクラスタへ収束する。

注意点：

  • 初期化次第で局所解に陥る → K-means++ 初期化、複数回試行
  • K は事前に決める → エルボー法、シルエット係数
  • 等方的・凸クラスタを仮定 → 非凸データには DBSCAN や階層クラスタが有効

§8 / PCA主成分分析（PCA）

高次元データを 分散最大方向 に射影して次元削減：

  1. データを中心化（平均を引く）
  2. 共分散行列 Σ = (1/n) XᵀX を計算
  3. Σ の固有値分解（章3）
  4. 上位 k 個の固有ベクトル（第1〜k主成分）に射影

固有値の大きさが「その方向の分散の量」。累積寄与率で次元数を決定。

FIG 8.5 PCAは「データの分散が最大の方向」を第一主成分とする。次元削減・可視化・ノイズ除去・特徴量抽出に多用される。

§9 / SUMMARY章のまとめ

1線形回帰は正規方程式で閉形式解
2Lasso（L¹）はスパース化、Ridge（L²）は重み縮小
3ロジスティック回帰：線形 + sigmoid + 交差エントロピー
4SVM：マージン最大化、カーネルトリックで非線形化
5決定木：解釈性高い、分岐基準は情報利得・ジニ
6ランダムフォレスト：バギング、特徴量サブセット
7GBDT：逐次的に誤差を埋める。XGBoost/LightGBM/CatBoost
8表データではいまだに GBDT が深層を凌駕することが多い
9K-means：割当・更新の反復、K は事前指定
10PCA：分散最大方向 = 共分散行列の固有ベクトル

章末問題

  1. 線形回帰の正規方程式を書けるか、XᵀX が非可逆だとどう困るか
  2. Lasso と Ridge の解析的・幾何的な違いを説明できるか
  3. ロジスティック回帰の損失（クロスエントロピー）が MLE と等価である理由
  4. SVM のマージン最大化を図で説明できるか
  5. RBF カーネルの式と「無限次元への写像」の意味
  6. 決定木の分岐基準（情報利得とジニ不純度）の式
  7. ランダムフォレストと GBDT の違い（並列 vs 逐次）
  8. K-means が局所解に落ちる例と、K-means++ の効果
  9. PCA がなぜ「共分散行列の固有値分解」なのか
  10. 表データで深層学習より GBDT が強い理由を説明できるか
