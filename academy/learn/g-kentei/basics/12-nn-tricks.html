§1 / BATCHミニバッチ学習

SGD のノイズを抑えつつ並列計算の恩恵を受けるため、m サンプル（典型 32〜256） ごとに平均勾配で更新するのが標準。

  • バッチサイズ小 → ノイジー、汎化が良い場合あり
  • バッチサイズ大 → 安定、並列効率良いが 「シャープミニマ」 に陥り汎化が落ちる傾向
  • 大規模学習では 線形スケーリング則：バッチサイズを k 倍にしたら学習率も k 倍

§2 / LR-SCHED学習率スケジューリング

FIG 12.1 主な学習率スケジューリング。Warmup（最初の数千ステップは低い学習率から徐々に上げる）は Transformer 学習で標準。
  • Step Decay：一定エポックごとに 1/10 など
  • Cosine Annealing：コサイン関数で滑らかに減衰
  • Warmup：最初の数ステップは小さい学習率から徐々に上げる（Transformer で標準）
  • ReduceLROnPlateau：検証ロスが改善しないとき学習率を下げる
  • One-Cycle：急上昇 → 急下降。fast.ai で普及

§3 / INIT重み初期化

不適切な初期化は学習が一切進まない原因に。

手法式用途
Xavier / GlorotVar(W) = 2 / (n_in + n_out)Sigmoid, Tanh
HeVar(W) = 2 / n_inReLU 系（標準）
LeCunVar(W) = 1 / n_inSELU
ゼロ初期化すべて 0NG（対称性が破れない）

§4 / NORMBatchNorm / LayerNorm

各層の入力を正規化（平均 0、分散 1 にスケール）してから学習可能なスケール・バイアスをかける：

y = γ · (x - μ) / √(σ² + ε) + β
FIG 12.2 BatchNorm vs LayerNorm。正規化の軸が違う。BatchNorm は CNN で標準、LayerNorm は Transformer や RNN で標準（バッチ非依存）。
正規化正規化軸主な用途
BatchNormバッチ方向CNN（バッチサイズが大きいとき）
LayerNorm特徴量方向（サンプルごと）RNN、Transformer（バッチ非依存）
GroupNormチャネルグループ方向バッチサイズが小さい CNN
InstanceNorm各サンプル、各チャネル画像スタイル変換
RMSNormLayerNorm の簡略化（平均引かない）Llama 系、近年の LLM

§5 / DROPOUTDropout

学習時に 各ユニットを確率 p（典型 0.5）で 0 にする 正則化手法（Hinton ら、2014）。

効果：

  • 暗黙のアンサンブル：毎ステップ違うサブネットを学習
  • ユニット間の共依存抑制
  • 推論時は無効化（あるいは出力を 1-p 倍）。これが inverted dropout

§6 / ADAMAdam とその仲間

Adam が事実上の標準。ただし用途によって以下も使われます：

Optimizer特徴主な用途
AdamMomentum + RMSProp。汎用標準大半の DL
AdamWAdam + 重み減衰の分離LLM 学習標準
SGD + Momentum古典、適切な学習率なら汎化が最良の場合画像分類の最終調整
LAMB大バッチ学習向けBERT 学習
LION2023年、メモリ効率良い、Adamに迫る性能大規模学習で台頭中
Sophia2次情報を効率的に活用LLM 学習で実験的

§7 / AUGデータ拡張

訓練データに 意味を変えない変形 を加えてサンプル数を増やす。暗黙の正則化でもある。

  • 画像：回転、反転、トリミング、色ジッタ、Mixup、CutMix、RandAugment
  • テキスト：言い換え、バックトランスレーション、トークン削除
  • 音声：時間引き伸ばし、ノイズ重畳、SpecAugment

§8 / OTHERその他の正則化・テクニック

  • Label Smoothing：one-hot を (1-ε, ε/(C-1), ..., ε/(C-1)) に。過信を抑える
  • Stochastic Depth：学習時にランダムに層をスキップ
  • Mixup：2つのサンプルを線形補間して新サンプル生成
  • EMA（指数移動平均）：重みの平滑版を保持、推論で使用
  • Gradient Accumulation：小バッチを複数回累積してから更新（GPUメモリ節約）
  • Mixed Precision：FP16/BF16 で計算、メモリと速度の改善

§9 / SUMMARY章のまとめ

1ミニバッチ学習（32〜256 が典型）が現代の標準
2大バッチには線形スケーリング則 + Warmup
3学習率スケジューリング：Step、Cosine、Warmup
4重み初期化：He（ReLU）、Xavier（Tanh）
5BatchNorm（CNN）vs LayerNorm（Transformer）の使い分け
6Dropout：暗黙のアンサンブル効果、推論時無効化
7Adam = 事実上の標準、AdamW = LLM の標準
8データ拡張は暗黙の正則化
9Label Smoothing、Mixup、EMA、Mixed Precision など現代の工夫多数
