「驚き」を、
数学化する。

サイコロで「1が出た」と「7が出た」では情報量が違います。前者は珍しくないが、後者は不可能 = 驚きが無限大。情報理論はこの直感を -log p(x) という数式に整理します。これがエントロピー・KL・クロスエントロピーの土台です。

§1 / INFO自己情報量：驚きの量

事象 x が確率 p(x) で起こるとき、その 自己情報量（驚き）は：

I(x) = -log p(x)
  • p(x) = 1（確実） → I(x) = 0（驚きなし）
  • p(x) = 0.5（コインの表） → I(x) = 1 ビット
  • p(x) → 0（極めて稀） → I(x) → ∞（最大の驚き）

底が 2 ならビット（情報量の単位）、底が e なら nat、底が 10 なら dit。機械学習では自然対数（nat）が標準。

FIG 6.1 自己情報量 I(x) = -log p(x)。確率が高い事象は驚きが小さく、確率が低い事象は驚きが大きい。これが「情報」を測る数学の出発点。

§2 / ENTROPYシャノンエントロピー

確率分布全体の 「不確かさ」 を、自己情報量の期待値として定義：

H(X) = -Σ p(x) log p(x) = E[I(X)]

性質：

  • H(X) ≥ 0
  • 1点集中（決定論的）→ H = 0（不確かさなし）
  • 一様分布で最大（最も不確か）
FIG 6.2 二値エントロピー H(p) = -p log p - (1-p) log(1-p)。p = 0.5（公平なコイン）で最大値 1 ビットに達し、p = 0 または 1 で 0 になる。
例題：エントロピーの計算
公平なサイコロ：H = -6 × (1/6) × log₂(1/6) = log₂ 6 ≈ 2.585 ビット
偏ったサイコロ（1の目だけ確率1）：H = -1 × log 1 = 0
4種のカードを等確率で引く：H = log₂ 4 = 2 ビット

§3 / KLKLダイバージェンス

2つの確率分布 P, Q の 「違い」 を測る量：

KL(P ‖ Q) = Σ p(x) log [p(x) / q(x)] = Σ p(x) [log p(x) - log q(x)]

性質：

  • KL(P ‖ Q) ≥ 0（Gibbs の不等式）
  • KL(P ‖ Q) = 0 ⟺ P = Q
  • 非対称：KL(P ‖ Q) ≠ KL(Q ‖ P)（厳密な「距離」ではない）
  • 三角不等式を満たさない
FIG 6.3 KL ダイバージェンスは2つの分布の違いを定量化。形が大きく違うほど KL は大きい。VAE 学習、強化学習の信頼領域、変分推論などで頻出。

§4 / CROSSクロスエントロピー

分類問題の損失関数の主役。正解分布 p（多くは one-hot）と予測分布 q に対して：

H(p, q) = -Σ p(x) log q(x) = H(p) + KL(p ‖ q)

つまり クロスエントロピー = 真の分布のエントロピー + KL。

機械学習では p（真の分布）は固定なので H(p) は定数。よって クロスエントロピーを最小化することは KL を最小化することと等価。これが分類で使われる理由です。

§5 / NLL分類損失の正体

正解が one-hot ベクトル（クラス k のみ 1）で、予測が softmax 出力 q のとき：

CE = -log q_k

つまり 「正解クラスに割り当てた確率の負対数」。

例題：3クラス分類のクロスエントロピー
正解: 猫（クラス2）。one-hot = (0, 1, 0)
予測 q = (0.2, 0.7, 0.1) → CE = -log 0.7 ≈ 0.357（良い予測、損失小）
予測 q = (0.6, 0.3, 0.1) → CE = -log 0.3 ≈ 1.204（悪い予測、損失大）
→ 正解クラスの確率が低いほど、損失が指数的に増える

章5 § 6.1 の最尤推定（MLE）と接続：分類の NLL（負対数尤度）= クロスエントロピー。同じものを別の見方で書いているだけ。

§6 / MI相互情報量

2つの確率変数 X, Y がどれだけ 独立から離れているか を測る：

I(X; Y) = KL(P(X,Y) ‖ P(X)P(Y))
  = H(X) - H(X | Y) = H(Y) - H(Y | X)

意味：「Y を知ったときに X の不確かさがどれだけ減るか」。
X と Y が独立なら I(X; Y) = 0。非線形な依存関係も検出できる（ピアソン相関は線形のみ）。

§7 / APPS機械学習での活用一覧

用途情報理論の道具
分類の損失関数クロスエントロピー
VAE の損失再構成誤差 + KL（事後と事前のずれ）
強化学習の信頼領域（TRPO/PPO）方策の更新を KL で制約
変分推論真の事後を近似する分布との KL 最小化
知識蒸留教師ソフト出力と生徒のクロスエントロピー
決定木の分岐基準情報利得 = 親エントロピー − 子エントロピー
言語モデル評価パープレキシティ = exp(クロスエントロピー)
非線形相関相互情報量

§8 / SUMMARY章のまとめ

1自己情報量 I(x) = -log p(x) は「驚き」の量
2シャノンエントロピー H(X) = E[-log p(X)] は分布の不確かさ
3一様分布で最大、決定論的分布で0
4KL ダイバージェンス：分布の違いの定量化（非対称・距離ではない）
5クロスエントロピー = 真の分布のエントロピー + KL
6分類損失は -log q_k（正解クラスの予測確率の負対数）
7クロスエントロピー最小化 = KL 最小化 = MLE と等価
8相互情報量：非線形な依存関係も検出
9VAE、PPO、変分推論、知識蒸留すべて KL の応用
10パープレキシティ = exp(CE) は言語モデルの標準評価指標
