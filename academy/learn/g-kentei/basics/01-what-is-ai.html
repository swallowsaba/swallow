章1 — AIとは何かChapter 1 — What is AI? Definitions and Thought Experiments

「AI」という言葉は曖昧です。本章では 知能の定義 を巡る哲学的議論を、図解とともに整理します。チューリングテスト、中国語の部屋、強いAI／弱いAI、AGI／ASI、フレーム問題、シンボルグラウンディング問題 — これらは70年前の論争ですが、ChatGPT を「理解している」と呼んでよいかという 2026年の議論にそのまま接続 します。

G-KENTEI · BASICS · CHAPTER 1

「知能」を、
定義する困難から始める

AI を理解する旅は、「知能とは何か」という問いから始まります。これは技術的な問いではなく、哲学的・認知科学的な問いです。だからこそ70年経っても決着していません。本章でその論点を整理することは、後続の全章（数学、機械学習、深層学習、生成AI）を 「これは結局、知能なのか単なる関数近似なのか」 という視点で読むための地盤になります。

§1 / DEFINITION知能を定義することの困難

「人工知能（Artificial Intelligence）」という言葉は、1956年のダートマス会議でジョン・マッカーシーが提唱しました。しかしマッカーシー自身、「知能」の厳密な定義は与えませんでした。知能という対象自体に普遍的定義がない 以上、その人工再現を定義することもできません。

この曖昧さは現代まで続き、AI研究者の間でも以下のような複数のアプローチが並立しています：

立場知能の定義代表的論者
機能主義知的に「振る舞え」ば知能である。内部状態は問わないチューリング、デネット
シンボル主義記号を操作する論理的能力こそ知能ニューウェル、サイモン
コネクショニズムニューロン的なネットワークが情報を処理することが知能マッカロック・ピッツ、ヒントン
身体性認知知能は身体と環境の相互作用から立ち上がるロドニー・ブルックス
強いAI批判意味の理解なしに振る舞いだけ似せても知能ではないジョン・サール

G検定では 各立場の主張を理解した上で、どれが正しいかという結論を出さない 姿勢が求められます。「正解」のない問いだということ自体が出題範囲です。

§2 / TURING TESTチューリングテスト

アラン・チューリングは1950年の論文 "Computing Machinery and Intelligence" で、知能の哲学的定義を避け、「機械が知的かどうかは、機械が知的に振る舞えるかで判定する」 という機能主義の発想を提案しました。これが チューリングテスト（オリジナルは「イミテーションゲーム」）です。

FIG 1.1  チューリングテストの構成。判定者 C は、テキストチャネルを通じてのみ A・B と会話する。一定回数の会話後、どちらが機械かを当てる。当てられなければ機械は「知的に振る舞った」と見なす。

テストの含意

  • 身体的特徴を排除：声・容姿は問わず、テキスト応答のみで判定する
  • 意味の理解は問わない：内部で何が起こっていてもよい、見える振る舞いが基準
  • 確率的判定：一度のテストではなく、多数の判定者の判断率で評価

批判

  • 欺瞞の能力に偏る：会話を成立させる能力 ≠ 全領域の知能
  • ELIZA（1966）など簡単なルールベースでも一部の人を欺けた
  • そもそも「人間に似せる」ことが知能の本質か？
  • 現代の LLM は表面的にはチューリングテストを 多くのケースで通過 するが、研究者の評価は分かれる
⚠ よくある誤解
「ChatGPT はチューリングテストに合格した」というニュースを目にすることがありますが、厳密に統制された大規模実験 はまだ確立されていません。短時間・限定設定での通過と、本格テストの通過は別物です。

派生：ローブナー賞

1991年から2019年まで、チューリングテストを実装した賞金コンテストが毎年開催されました（ヒュー・ローブナーが資金提供）。本格賞（人間と区別不能なAI）には誰も到達できず、年ごとに「最も人間らしい」AI に銅メダルが授与されるだけで終わりました。

§3 / CHINESE ROOM中国語の部屋

機能主義（振る舞いで判定する）への最も鋭い反論が、哲学者 ジョン・サール が1980年に提示した 「中国語の部屋」 という思考実験です。

FIG 1.2  中国語の部屋。中の人は英語話者で中国語は一切分からないが、ルール集を引いて応答する。外から見ると中国語の自然な会話に見える — でも内部に「理解」はあるか？

サールの主張

  1. シンタックス（記号操作）はセマンティクス（意味）を生まない
  2. コンピュータは記号を機械的に操作しているだけ
  3. 従って、どれほど見事に「振る舞い」を再現しても、それは「強いAI（真の理解を持つ）」ではない
  4. LLM もこの議論の範囲：流暢な応答 ≠ 理解

反論

  • システム反論：理解しているのは「中の人」ではなく システム全体（部屋 + ルール集 + 人）である
  • ロボット反論：実世界とつながった身体性があれば、記号は意味に接地する（→ シンボルグラウンディング問題）
  • 創発反論：理解は十分複雑なシステムから自然に立ち上がる性質ではないか
→ 現代への接続
LLM のハルシネーション（事実と異なる流暢な出力）は、サールの主張の現代版とも読めます。「言葉を意味と接地せず、統計的に最も尤もらしい次トークンを出している」だけだとすれば、中国語の部屋の現代版です。詳細は 応用編。

§4 / STRONG vs WEAK強いAI / 弱いAI

サールが導入したもう一つの重要概念が、強いAI / 弱いAI の区別です。

FIG 1.3  弱いAI と 強いAI の対比。現在実用化されているAIはすべて「弱いAI」のカテゴリ。強いAI は技術的にも哲学的にも未到達。
弱いAI強いAI
定義特定の知的タスクをこなす道具人間と同等の意識・主観・理解を持つ存在
実例画像分類器、機械翻訳、ChatGPT、AlphaGo（全て）未実現
必要なものうまく動くアルゴリズムとデータ意識の科学的解明（未解明）
判定基準タスク性能判定不能（外から測れない）

重要な注意：「弱い」は性能の低さを意味しません。AlphaGo は囲碁で人間を超えていますが、弱いAI です。「弱い」とは「知能の一側面に特化している」という意味です。

§5 / ANI/AGI/ASIANI / AGI / ASI のスペクトル

「弱いAI / 強いAI」が哲学的区別なのに対し、ANI / AGI / ASI は能力範囲による区別です。スペクトルとして捉えるのが現代の見方です。

FIG 1.4  ANI／AGI／ASI のスペクトル。現在は ANI を全面実用化し、AGI の入口に立っているとされる。ASI は仮説段階。

近年の論争：LLM は AGI か？

2023年以降、GPT-4 や Claude などの大規模言語モデルが、複数領域でタスクをこなすことから、「AGI に近づいた」という主張が出ています。

  • 肯定派：法律・医療・コーディングなど多領域でプロ並みのスコア。「Sparks of AGI」（Microsoft Research, 2023）
  • 否定派：計画立案・継続学習・身体性が欠落。記憶も持たない。AGI とは呼べない
  • 中立派：定義次第。「AGI」自体に合意がない

§6 / FRAME PROBLEMフレーム問題

1969年、ジョン・マッカーシーとパトリック・ヘイズが提起した古典問題。「AIは、ある行動が世界の何を変え、何を変えないかを、どう知ればよいか」 という問題です。

FIG 1.5  フレーム問題のロボット例。単純な「電池を取る」というタスクでも、世界のどの側面を考慮し、どれを無視するかを有限ステップで決定することが論理的に困難。これがシンボリックAIの根本的限界。

本質

フレーム問題は 「実世界では考慮事項が事実上無限にある」ことから来ます。AIが「いま考えるべきことのフレーム（枠）」を有限ステップで選び出せない。

古典AIではこれが致命的な制約でした。現代の機械学習は、フレームを 明示的にプログラムせず、データから学ぶ ことで部分的に回避していますが、根本的な解決には至っていません。LLM の文脈長の制限や、未知の状況での失敗は、フレーム問題の現代版とも言えます。

§7 / SYMBOL GROUNDINGシンボルグラウンディング問題

1990年、認知科学者スティーブン・ハルナッドが提起した問題。「記号は、どうやって実世界の意味に結びつくのか」。

FIG 1.6  シンボルグラウンディング問題。AIの内部にある「リンゴ」という記号（または埋め込みベクトル）は、実世界のリンゴという物体とどう結びついているか？ 結びついていないなら、AI は本当に「リンゴ」を理解しているのか？

含意と現代への接続

LLM は膨大なテキストから学習しますが、身体を持たず、実世界の経験を持ちません。「リンゴ」というトークンは他の単語との共起パターンとしてのみ存在します。これがハルシネーションの一因と主張する研究者も多くいます。

近年の対応：

  • マルチモーダル化：CLIP、GPT-4V、Gemini は画像と言語を結びつけることで、ある程度の接地を達成
  • 身体性AI：ロボティクスと言語モデルの融合（NVIDIA Eureka、Google RT-2）
  • RAG：外部知識ベースで事実性を補強

§8 / NO FREE LUNCHノーフリーランチ定理

David Wolpert が1996年に証明した数学的定理。「すべての問題に対して平均的にみると、どの最適化アルゴリズムも同等の性能」。

∀ アルゴA, アルゴB： 全問題上での平均性能(A) = 全問題上での平均性能(B)

含意

  • 万能のアルゴリズムは存在しない。深層学習も例外ではない
  • あるアルゴリズムが問題 X で優れているのは、その問題の構造に合った帰納バイアス を持っているから
  • 新しい問題ではアルゴリズム選択が重要。常に同じものを使うのは合理的でない
  • 「Attention Is All You Need」は 「言語系列の問題には」 という前提付きの主張

この定理が示すのは、「AI研究は問題ごとの帰納バイアス設計の科学」 という見方です。CNN は画像の局所性、Transformer は系列の柔軟な依存関係、GNN はグラフ構造、というように、それぞれが特定構造に最適化されています。

§9 / SINGULARITYシンギュラリティの議論

未来学者 レイ・カーツワイル が2005年の著書 『The Singularity Is Near』（邦題『ポスト・ヒューマン誕生』）で提示した予測。「2045年頃に技術的特異点（Technological Singularity）が到来し、それ以降は人間に予測不能な速度で技術が進歩する」。

FIG 1.7  カーツワイルのシンギュラリティ仮説。AIの能力は指数関数的に伸び、ある時点で人間を超え、その後は人間が予測できない速度で進化するという見立て。あくまで仮説であり、現実がこのように進むかは不明。

反論

  • 指数的進歩は永遠に続かない（物理的・経済的制約）
  • 過去の予測（核融合、量子コンピュータ）は常に「20年後」のままだった
  • 「知能」が単一の量として表せるかが疑問
  • 議論の経済的インセンティブ（恐怖は売れる）

G検定では、シンギュラリティ自体の真偽を問わず、概念とその提唱者を知っていることが求められます。社会的影響として「労働の変容」「教育の変革」「倫理問題」と結びつけて議論されます。

§10 / SUMMARY章のまとめ

本章で覚えるべき10の核概念

1チューリングテスト — 機能主義に基づく知能の判定。テキスト応答で人間と区別不能なら知的
2中国語の部屋 — サールの反論。シンタックス操作は意味を生まない
3強いAI / 弱いAI — 哲学的区別。意識を持つかどうか
4ANI / AGI / ASI — 能力範囲の区別。現在は ANI のみ実現
5フレーム問題 — 何を考慮し何を無視するかを有限ステップで決定できない
6シンボルグラウンディング問題 — 記号が実世界の意味に結びつかない
7ノーフリーランチ定理 — 万能のアルゴリズムは存在しない。帰納バイアスが必要
8シンギュラリティ — カーツワイルの2045年予測。技術的特異点
9身体性認知 — 知能は身体と環境の相互作用から立ち上がるという立場
10LLM 時代の議論 — これらの古典問題は現代生成AIにも姿を変えて現れる
