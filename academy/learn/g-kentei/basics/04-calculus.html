損失の地形を、
下りていく科学。

機械学習は「データを入力する → モデルが予測する → 損失を計算する → 損失を減らす方向にパラメータを動かす」というループの繰り返し。「損失を減らす方向」を計算するのが微分 で、動かすのが最適化アルゴリズム。本章でこの2つを根本から理解します。

§1 / DERIVATIVE微分：変化率

関数 f(x) の 微分 f'(x) は、x が微小に変化したときの f の変化率：

f'(x) = lim_{h→0} [f(x+h) - f(x)] / h

幾何学的には 接線の傾き。f が急に増加している場所は微分が大きく、変化していない場所は微分が 0。

FIG 4.1 微分の幾何学的意味は「接線の傾き」。最小点では傾き 0、急な勾配ではその点での傾きが大きい。これが最適化の鍵。

§2 / PARTIAL偏微分

多変数関数 f(x₁, x₂, ..., xₙ) を、特定の変数 xᵢ だけについて 微分したものが 偏微分 ∂f/∂xᵢ。他の変数は定数扱い。

例：f(x, y) = x²y + 3xy²
∂f/∂x = 2xy + 3y²   (y を定数扱い)
∂f/∂y = x² + 6xy   (x を定数扱い)

§3 / GRADIENT勾配ベクトル：最急上昇方向

多変数関数 f の 勾配 ∇f は、偏微分を全変数について並べた列ベクトル：

∇f(x) = (∂f/∂x₁, ∂f/∂x₂, ..., ∂f/∂xₙ)ᵀ

核心的意味：勾配 ∇f は「その点で f が最も急に増加する方向」を指す。
→ 反対方向 -∇f は最も急に減少する方向。これが勾配降下法の根拠。

FIG 4.2 勾配 ∇f は等高線に直交し、最も急な上昇方向を指す。-∇f は最急降下方向。機械学習はこの方向にパラメータを動かして損失を減らす。

§4 / CHAIN連鎖律

合成関数 f(g(x)) の微分：

d/dx [f(g(x))] = f'(g(x)) · g'(x)

これがバックプロパゲーションの数学的基盤。多層NNでは合成関数が何層にも重なるが、連鎖律を機械的に適用すれば全パラメータの勾配が計算できる。

例：f(g(x)) = (3x + 1)²
g(x) = 3x + 1, f(u) = u²
g'(x) = 3, f'(u) = 2u
d/dx [(3x+1)²] = 2(3x+1) · 3 = 6(3x+1)

§7 / GD勾配降下法

損失 L(θ) を最小化したい。パラメータ θ を、勾配の逆方向に少しずつ動かす：

θ_{t+1} = θ_t − η · ∇L(θ_t)

η は 学習率（learning rate）。これが最適化の最重要ハイパーパラメータ。

FIG 4.4 勾配降下法のトラジェクトリ（適切な学習率）。各ステップで -∇L 方向に進み、最小点に近づく。

§8 / LR学習率の影響

FIG 4.5 学習率の選択：小さすぎると遅く、大きすぎると発散する。ニューラルネット学習で最も重要なハイパーパラメータ。
例題：1次元の勾配降下を手で1ステップ
L(θ) = (θ - 3)², 初期 θ₀ = 0, η = 0.1
∇L(0) = 2(0-3) = -6
θ₁ = 0 - 0.1 × (-6) = 0.6
θ₂ = 0.6 - 0.1 × (2(0.6-3)) = 0.6 - 0.1 × (-4.8) = 1.08
... 反復すると θ → 3（最小点）に収束
⚠ 学習率を大きくし過ぎると
η = 1.5 で同じ問題：θ₁ = 0 - 1.5 × (-6) = 9。次に θ₂ = 9 - 1.5 × 12 = -9。発散（振動拡大）。
