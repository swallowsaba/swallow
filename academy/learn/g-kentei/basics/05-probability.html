章5 — 確率・統計Chapter 5 — Probability & Statistics

機械学習は、本質的に 不確実性を扱う学問 です。ラベルにはノイズが乗り、データはサンプル、モデルは確率的予測を出す。本章では、確率の公理 から、条件付き確率、ベイズの定理、確率変数と分布、期待値・分散・共分散、主要な分布、大数の法則・中心極限定理、そして 最尤推定（MLE） と ベイズ推定（MAP） までを5部構成で徹底解説します。クロスエントロピー損失、ナイーブベイズ、ガウス混合モデル（GMM）、VAE まで、確率の言葉を母語にすることが目標です。

不確実性を、
厳密に扱う言葉。

『この画像は犬である確率は 87%』── これは何を意味しているでしょう。87%という数字はどこから来て、どう信頼すべきか。確率論はその枠組みを与えます。コルモゴロフの3公理から始まり、条件付き確率、ベイズの定理、確率変数、分布、期待値、中心極限定理、最尤推定、ベイズ推定 ── これらが 機械学習の確率的解釈 の根幹です。NN の出力を softmax で確率にし、クロスエントロピーで学習し、不確実性を Monte Carlo Dropout で推定する ── すべて、本章の言葉で記述されます。

SECTION ONE

基本

確率公理・条件付き確率・ベイズ・確率変数・期待値分散・離散連続分布・CLT・MLE・MAPの10節。

基本

§1.1 確率の公理・条件付き確率・独立性

コルモゴロフの3公理が現代確率論の基礎：

① 非負性：P(A) ≥ 0
② 全確率：P(Ω) = 1（Ω は全事象）
③ 加法性：互いに排反な事象列 Aᵢ について P(∪Aᵢ) = ΣP(Aᵢ)

条件付き確率　 P(A|B) = P(A∩B) / P(B)　 (P(B) > 0)
独立性　 P(A∩B) = P(A) P(B)　 ⇔ 　 P(A|B) = P(A)
FIG 5.1 ／ 確率の公理と条件付き確率。P(A|B) は『B が起きた』という情報を加味した A の確率。独立性は確率モデルの強い仮定で、ナイーブベイズの根幹。

§1.2 ベイズの定理 — 信念の更新

ベイズの定理　 P(A|B) = P(B|A) P(A) / P(B)

　 P(A)　 = 事前確率（prior）
　 P(B|A) = 尤度（likelihood）
　 P(A|B) = 事後確率（posterior）
　 P(B)　 = エビデンス（規格化定数）

機械学習の文脈　 P(θ|D) ∝ P(D|θ) P(θ)
　 観測 D 後のパラメータ分布 ∝ 尤度 × 事前
▎ 疾病検査の例（古典的、しかし重要）
ある疾病の有病率が 1%（P(病気)=0.01）。検査は感度 99%（P(陽性|病気)=0.99）、特異度 99%（P(陰性|健康)=0.99）。検査陽性のとき、本当に病気である確率は？

ベイズの定理：
P(病気|陽性) = P(陽性|病気)P(病気) / P(陽性)
P(陽性) = P(陽性|病気)P(病気) + P(陽性|健康)P(健康) = 0.99×0.01 + 0.01×0.99 = 0.0198
P(病気|陽性) = 0.99×0.01 / 0.0198 ≈ 50%

『陽性なら 99% 病気』と思いがちだが、有病率が低いと実は半々。直感を裏切るのでベイズ的思考が重要。
FIG 5.2 ／ ベイズの定理は『信念の更新』。事前分布 P(θ) を尤度 P(D|θ) で更新して事後分布 P(θ|D) を得る。次の観測時にはこれを新たな事前として使え、オンライン学習に自然。

§1.3 確率変数と分布

確率変数 X：標本空間 Ω から実数への写像。出る値が確率的に決まる量。離散と連続で扱いが分かれる：

  • 離散確率変数：とりうる値が有限or可算無限（サイコロ、コイン）。確率質量関数 PMF：P(X=x)。
  • 連続確率変数：実数全体や区間（身長、温度）。確率密度関数 PDF：f(x) で、P(a<X<b) = ∫f(x)dx。
  • 累積分布関数 CDF：F(x) = P(X≤x)。離散・連続共通。
▎ PDF の値は確率ではない
連続変数では『特定の値 x をとる確率』は 0。PDF f(x) は確率密度であり、1 を超えることもある（例：N(0, 0.1²) の x=0 で値 ≈ 3.99）。確率は積分でしか得られない。

§1.4 期待値・分散・共分散

期待値　 E[X] = Σ x P(X=x)　 （離散）　 or 　 ∫ x f(x) dx　 （連続）
分散　 Var(X) = E[(X - E[X])²] = E[X²] - E[X]²
標準偏差　 σ = √Var(X)
共分散　 Cov(X, Y) = E[(X - E[X])(Y - E[Y])]
相関係数　 ρ(X, Y) = Cov(X, Y) / (σ_X σ_Y) ∈ [-1, 1]

重要な性質
　 E[X + Y] = E[X] + E[Y] （線形性、常に成立）
　 Var(aX + b) = a² Var(X)
　 X, Y 独立 ⇒ Var(X + Y) = Var(X) + Var(Y)

§1.5 主要な離散分布

分布記法意味E[X]Var(X)
ベルヌーイBe(p)1回の試行の成功（1）／失敗（0）pp(1-p)
二項Bin(n, p)n回試行の成功回数npnp(1-p)
ポアソンPoi(λ)単位時間あたり平均λ回の希少事象数λλ
幾何Geo(p)初めて成功するまでの試行数1/p(1-p)/p²
多項Mult(n, p)K面サイコロをn回振った各面の回数npᵢnpᵢ(1-pᵢ)

§1.6 主要な連続分布

分布記法用途E[X]Var(X)
一様U(a, b)区間内で等確率(a+b)/2(b-a)²/12
正規N(μ, σ²)誤差・身長・標本平均の極限μσ²
指数Exp(λ)イベント間時間1/λ1/λ²
ガンマΓ(α, β)正の連続量、共役事前α/βα/β²
ベータBeta(α, β)[0,1] 上、ベルヌーイの共役α/(α+β)αβ/((α+β)²(α+β+1))
カイ二乗χ²(k)仮説検定、分散の分布k2k
t分布t(ν)小サンプルの t検定0ν/(ν-2)
FIG 5.3 ／ 機械学習で頻出の3つの連続分布。正規（誤差、CLT）、指数（生存時間）、ベータ（[0,1]、ベイズで共役）。

§1.7 多変量正規分布

多変量正規分布 N(μ, Σ)
　 p(x) = (2π)-d/2 |Σ|-1/2 exp(-½ (x-μ)TΣ⁻¹(x-μ))

　 μ ∈ ℝd　 平均ベクトル
　 Σ ∈ ℝd×d　 共分散行列（対称・半正定値）

性質　 線形変換しても正規分布、周辺分布も条件付き分布も正規分布

§1.8 大数の法則と中心極限定理

大数の法則（LLN）　 X₁, X₂, ... が i.i.d. で E[X]=μ なら
　 X̄n = (1/n)Σ Xᵢ → μ 　（n→∞ で確率収束）

中心極限定理（CLT）　 さらに Var(X)=σ² < ∞ なら
　 √n(X̄n - μ) / σ → N(0, 1)
　 母集団分布が何であれ、標本平均は正規分布に近づく
FIG 5.4 ／ 中心極限定理。元の分布が何であれ、標本平均は n が大きくなると正規分布に収束する。信頼区間・仮説検定・A/B テスト・ミニバッチSGDの根拠。

§1.9 最尤推定（MLE）

最尤推定　 観測 D = {x₁, ..., x_n} を最も生成しやすい θ を選ぶ
　 θ̂MLE = argmaxθ Π p(xᵢ|θ) = argmaxθ Σ log p(xᵢ|θ)

機械学習との対応
　 線形回帰の MLE（等分散正規ノイズ） ≡ OLS（最小二乗法）
　 ロジスティック回帰の MLE ≡ クロスエントロピー損失最小化
　 NN の MLE ≡ 多項分布尤度 ≡ CE 損失

§1.10 ベイズ推定 — MAP と事後分布

MAP 推定（Maximum A Posteriori）
　 θ̂MAP = argmaxθ P(θ|D) = argmaxθ P(D|θ) P(θ)
　 事前 P(θ) が一様 ⇒ MAP = MLE

正則化の確率的解釈
　 L2 正則化 ≡ ガウス事前 P(θ) ∝ exp(-λ||θ||²) の MAP
　 L1 正則化 ≡ ラプラス事前 P(θ) ∝ exp(-λ||θ||₁) の MAP

完全なベイズ推定　 事後分布 P(θ|D) 全体を扱う（不確実性も含む）
FIG 5.5 ／ MLE は尤度のピーク、MAP は事前×尤度のピーク、完全なベイズ推定は事後分布全体を扱い不確実性も推定する。MCMC、変分推論はベイズ推定の実装手段。
SECTION TWO

応用

ナイーブベイズ、ロジ回帰の確率解釈、GMMとEM、変分推論とVAEの4節。

応用

§2.1 ナイーブベイズ — 独立性仮定の威力

テキスト分類などで使われる確率的分類器。特徴量が条件付き独立と仮定して計算を劇的に単純化：

P(クラス | x₁, x₂, ..., x_d) ∝ P(クラス) · Π P(xᵢ | クラス)
　　　　　　　　　　　　　　　　　　　 ↑ 独立性仮定で積に

仮定は『naive（素朴）』だが、実用では十分機能することが多く、スパムフィルタの古典的手法。多項分布版がテキスト分類、ガウス版が連続特徴量に。

§2.2 ロジスティック回帰の確率的解釈

シグモイド出力 σ(w·x + b) を P(y=1|x) とみなし、ベルヌーイ尤度の MLE で学習：

負の対数尤度（クロスエントロピー損失）
　 L(w) = -Σᵢ [yᵢ log σ(zᵢ) + (1-yᵢ) log(1-σ(zᵢ))]
　 　　= ベルヌーイ尤度の負の対数

§2.3 ガウス混合モデル（GMM）と EM アルゴリズム

K 個の多変量正規分布の重み付き和でデータを表現：

　 p(x) = Σk=1K πk N(x | μk, Σk)　 (Σπₖ = 1)

EM アルゴリズム　 潜在変数を含むモデルの MLE 反復解法
　 E-step：現パラメータで潜在変数の事後（責任度）を計算
　 M-step：それを使って ELBO を最大化するパラメータ更新

§2.4 変分推論と VAE

事後分布 P(θ|D) が解析的に求まらない場合、扱いやすい分布族 q(θ;φ) で近似し ELBO 最大化：

ELBO（Evidence Lower BOund）
　 ELBO = Eq[log P(D, θ)] - Eq[log q(θ;φ)]
　 log P(D) = ELBO + KL(q || P(θ|D))

VAE　 Encoder が q(z|x) を、Decoder が p(x|z) を学習。損失 = -ELBO

ハンズオン

§3.1 ベイズの定理で疾病検査

# 有病率 1%、感度 99%、特異度 99% の検査 P_disease = 0.01 P_pos_given_disease = 0.99 P_pos_given_healthy = 0.01 # 1 - 特異度 # 全確率の定理 P_pos = (P_pos_given_disease * P_disease + P_pos_given_healthy * (1 - P_disease)) # ベイズの定理 P_disease_given_pos = P_pos_given_disease * P_disease / P_pos print(f"陽性のとき病気である確率: {P_disease_given_pos:.4f}") # 0.5000 ← 直感に反する # 有病率を 20% に上げると P_disease = 0.20 P_pos = 0.99 * 0.20 + 0.01 * 0.80 P_disease_given_pos = 0.99 * 0.20 / P_pos print(f"有病率20%なら: {P_disease_given_pos:.4f}") # 0.9612
OUTPUT
陽性のとき病気である確率: 0.5000
有病率20%なら: 0.9612

§3.3 MLE のスクラッチ — コインの公平性

# 10回コインを投げて 7回表が出た # 表の確率 p の最尤推定 n_heads = 7 n_total = 10 # 尤度 L(p) = C(n,k) p^k (1-p)^(n-k) # 対数尤度の微分 = 0 → p = k/n p_mle = n_heads / n_total print(f"MLE: p = {p_mle:.3f}") # 0.700 # 数値的に確認 from scipy.optimize import minimize_scalar def neg_log_lik(p): return -(n_heads * np.log(p) + (n_total - n_heads) * np.log(1-p)) res = minimize_scalar(neg_log_lik, bounds=(0.01, 0.99), method='bounded') print(f"数値最適化: p = {res.x:.3f}") # MAP 推定：Beta(α=2, β=2) 事前（公平側に弱い偏り） alpha, beta = 2, 2 p_map = (n_heads + alpha - 1) / (n_total + alpha + beta - 2) print(f"MAP (Beta(2,2)事前): p = {p_map:.3f}") # 0.643
OUTPUT
MLE: p = 0.700
数値最適化: p = 0.700
MAP (Beta(2,2)事前): p = 0.643
（事前『多分公平』のおかげで 0.5 に引き寄せられている）

§3.5 Bayesian A/B テスト

from scipy.stats import beta # A: 1000 訪問のうち 50 コンバージョン # B: 1000 訪問のうち 65 コンバージョン # ベータ事前 + ベルヌーイ尤度 = ベータ事後 alpha_A, beta_A = 1 + 50, 1 + 950 alpha_B, beta_B = 1 + 65, 1 + 935 # 事後からサンプリングして P(B が A より良い) samples_A = beta.rvs(alpha_A, beta_A, size=100000) samples_B = beta.rvs(alpha_B, beta_B, size=100000) p_B_better = (samples_B > samples_A).mean() print(f"P(B が A より良い) = {p_B_better:.4f}") # 約 0.87 → かなり信頼できる差 # 期待改善率 lift = (samples_B - samples_A) / samples_A print(f"期待リフト = {lift.mean()*100:.2f}%") print(f"95%信用区間 = ({np.percentile(lift, 2.5)*100:.1f}%, " f"{np.percentile(lift, 97.5)*100:.1f}%)")

コインの公平性 p について、観測ごとに信念がどう更新されるか。

SIM 3 — モンテカルロ積分

π の値を確率的方法（ランダム点を撃つ）で推定。確率の応用の典型例。
>> [-1, 1]² の正方形にランダムに点を撃つ
>> 半径 1 の円内の点の割合 × 4 ≈ π

n=100:　　π ≈ 3.04　 誤差 ±0.10
n=1000:　 π ≈ 3.156　誤差 ±0.015
n=10000:　π ≈ 3.140　誤差 ±0.0014
n=100000: π ≈ 3.1418 誤差 ±0.0001

>> 真値 π ≈ 3.14159
>> 誤差は 1/√n で減少（CLT）
>> これがモンテカルロ積分の理論的根拠
SECTION FIVE

問題

G検定形式の演習問題 20問。確率・統計のすべての概念を確認。

問題

PROBLEM 1
サイコロを2回振って和が 7 になる確率は。
① 1/36　 ② 1/12　 ③ 1/6　 ④ 1/4
ANSWER ③ 和が7になる組合せは (1,6)(2,5)(3,4)(4,3)(5,2)(6,1) の6通り、全 36 通り → 6/36 = 1/6。
PROBLEM 2
事象 A と B が独立であることの定義として正しいのは。
① P(A∪B) = P(A) + P(B)　 ② P(A∩B) = P(A)P(B)　 ③ A=B　 ④ P(A|B) = 0
ANSWER ② 独立性の定義は P(A∩B) = P(A)P(B)。これは P(A|B) = P(A) と同値。①は『排反』ではなく『独立』との混同に注意。
PROBLEM 3
ベイズの定理 P(A|B) = ? を正しく表すのは。
① P(B|A) P(A) / P(B)　 ② P(A) P(B) / P(A∩B)　 ③ P(A∩B) / P(A)　 ④ P(A) + P(B)
ANSWER ① ベイズの定理。これにより事前 × 尤度 / エビデンスで事後分布が得られる。
PROBLEM 4
連続確率変数の確率密度関数 f(x) が満たすべき性質は。
① 0 ≤ f(x) ≤ 1　 ② f(x) ≥ 0 かつ ∫f(x)dx = 1　 ③ f(x) = 1　 ④ 離散値のみ
ANSWER ② PDFは非負＋全積分1。値自体は1を超えうる（密度であり確率ではない）ことに注意。
PROBLEM 5
確率変数 X, Y が独立のとき Var(X+Y) は。
① Var(X) + Var(Y)　 ② Var(X) - Var(Y)　 ③ Var(X)Var(Y)　 ④ Var(XY)
ANSWER ① 一般に Var(X+Y) = Var(X) + Var(Y) + 2Cov(X,Y)。独立なら Cov = 0 で右辺第3項が消える。
PROBLEM 6
ベルヌーイ分布 Be(p) の平均と分散は。
① p, p²　 ② p, p(1-p)　 ③ 1-p, p　 ④ 0, p
ANSWER ② E[X]=p, Var[X]=p(1-p)。分散は p=0.5 で最大 0.25。
PROBLEM 7
標準正規分布 N(0, 1) の確率密度関数の値が最大となる x は。
① -1　 ② 0　 ③ 1　 ④ ∞
ANSWER ② 正規分布のピークは平均 μ=0 の位置。最大密度は 1/√(2π) ≈ 0.399（1を超えない例）。
PROBLEM 8
中心極限定理の主張として正しいのは。
① 母集団は必ず正規　 ② 標本平均は母集団分布によらず正規分布に近づく　 ③ 期待値=分散　 ④ どんな分布も同じ平均
ANSWER ② CLT は母集団分布が何であれ（有限分散なら）、標本平均が正規分布に収束することを示す。実用上、n≥30 で近似が良い。
PROBLEM 9
最尤推定（MLE）の基本的な考え方は。
① 観測データを最も尤もらしくするパラメータを選ぶ　 ② 事前分布の中央値　 ③ 事後分布の期待値　 ④ ランダム選択
ANSWER ① MLEは観測データの尤度 P(D|θ) を最大化する θ を選ぶ。実装では対数尤度を最大化（数値安定）。
PROBLEM 10
MAP 推定と MLE の関係は。
① 完全に同じ　 ② 事前分布が一様なら一致　 ③ MAP は事前を使わない　 ④ MLEはベイズ
ANSWER ② MAP = argmax P(D|θ)P(θ)、事前が一様なら P(θ) が定数となり MAP=MLE。L2 正則化はガウス事前のMAPに相当。
PROBLEM 11
機械学習で『L2 正則化』に対応する事前分布は。
① 一様　 ② ガウシアン　 ③ ラプラス　 ④ ベータ
ANSWER ② L2 ≡ -log P(θ) で P(θ) ∝ exp(-λ||θ||²) なのでガウシアン。L1 はラプラス分布の MAP に対応。
PROBLEM 12
ナイーブベイズが『naive（素朴）』と呼ばれる理由は。
① パラメータが少ない　 ② 特徴量が条件付き独立と仮定する　 ③ 線形　 ④ 学習が速い
ANSWER ② 強い独立性仮定が『素朴』の由来。実際にはほとんど成立しないが、結果は意外と頑健で実用的。
PROBLEM 13
EM アルゴリズムの典型用途は。
① 線形回帰の解析解　 ② 潜在変数を含むモデルの MLE（GMM など）　 ③ 勾配計算　 ④ 正則化選択
ANSWER ② EM は潜在変数（クラスタ割当、隠れ状態）を含むモデルの反復 MLE。GMM, HMM, LDA, VAE の基礎。
PROBLEM 14
変分推論（VI）が最大化する量は。
① 尤度のみ　 ② ELBO（Evidence Lower BOund）　 ③ クロスエントロピー　 ④ 距離
ANSWER ② VIは事後分布 P(θ|D) を近似分布 q(θ;φ) で近似し、ELBO を最大化する。これは KL(q||P) 最小化と等価。VAE の損失の根拠。
PROBLEM 15
95% 信頼区間の正しい解釈は。
① 真値が 95% の確率でこの区間にある　 ② 同じ手順を多数回繰り返すと 95% の区間が真値を含む　 ③ 必ず真値を含む　 ④ 効果量
ANSWER ② 頻度論の解釈は『同じ手順を繰り返すと』。①はベイズの『信用区間』の解釈で、頻度論では誤り。
PROBLEM 16
p値の正しい意味は。
① H0 が真の確率　 ② H0 が真と仮定したとき、観測以上に極端な結果が得られる確率　 ③ H1 が真の確率　 ④ 効果量
ANSWER ② p値は条件付き確率 P(データ|H0)。『H0 が真の確率』ではない（よくある誤解）。p < α で H0 棄却。
PROBLEM 17
多変量正規分布 N(μ, Σ) において Σ が満たすべき条件は。
① 対角行列　 ② 対称かつ半正定値　 ③ 単位行列　 ④ 上三角
ANSWER ② 共分散行列は常に対称、必ず半正定値。これは共分散の定義から数学的に従う。
PROBLEM 18
分類器の確率出力（softmax）が真の確率と一致するように調整する手法を何と呼ぶか。
① キャリブレーション（Platt Scaling、Isotonic、Temperature Scaling など）　 ② 正則化　 ③ 標準化　 ④ 量子化
ANSWER ① 現代のNNは過信気味で、出力確率と真の頻度が一致しないことが多い。Calibration がそれを補正。Reliability Diagram, ECE で評価。
PROBLEM 19
ベイズ的 A/B テストの利点は。
① p値だけが分かる　 ② 『B が A より良い確率』など直観的な確率解釈ができ、逐次更新が自然　 ③ サンプル不要　 ④ 必ず A が勝つ
ANSWER ② 事後分布から『P(B>A)』が直接出る。逐次的なデータ追加で事前を更新でき、peeking 問題が起きにくい。
PROBLEM 20
多重検定の補正で最も保守的な手法は。
① Bonferroni（α/m）　 ② FDR (Benjamini-Hochberg)　 ③ Holm　 ④ 無補正
ANSWER ① Bonferroni は α/m と最も厳しい。検出力は犠牲だが、FWER（少なくとも1つの偽発見の確率）を α 以下に保証。現代の標準は FDR（Benjamini-Hochberg）。

章のまとめ

確率の公理から始め、条件付き確率・ベイズの定理（信念の更新）、確率変数と分布、期待値・分散・共分散、主要分布（ベルヌーイ・二項・ポアソン・正規・ベータ・ガンマ）、多変量正規分布、大数の法則と中心極限定理、最尤推定 MLE、ベイズ推定 MAP までを扱いました。応用ではナイーブベイズ、ロジスティック回帰の確率解釈、GMM と EM、変分推論と VAE。ハンズオンでベイズ計算・CLT 実験・MLE・GMM・Bayesian A/B、シミュレーションで CLT・ベイズ更新・モンテカルロ積分。次章は情報理論。エントロピー、相互情報量、KLダイバージェンスを通じて、機械学習の損失関数の意味が明らかになります。
