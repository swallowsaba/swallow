章5 — 確率と統計Chapter 5 — Probability and Statistics

機械学習は 不確かさを扱う数学 です。「このメールはスパムか？」「このX線画像は癌か？」のような予測は本質的に確率的。確率分布、中心極限定理、ベイズの定理、最尤推定 を直感と式の両面から押さえます。

不確かさを、
数式で扱う。

現実の AI システムは決定論的ではなく、すべて確率的に動作します。「90% の確信度で犬」「事後確率 0.7 でこの治療が有効」というように。本章では確率の道具立てを揃え、機械学習の損失関数（クロスエントロピー）が 最尤推定の自然な帰結 であることまで進みます。

§1 / RANDOM確率変数

確率変数 X は「値が確率的に決まる変数」。離散と連続の2種類があります。

離散：確率質量関数（PMF） P(X = x)。サイコロなら P(X=1) = ... = P(X=6) = 1/6。

連続：確率密度関数（PDF） f(x)。値そのものではなく 積分 が確率を表す：

P(a ≤ X ≤ b) = ∫ₐᵇ f(x) dx,   ∫₋∞^∞ f(x) dx = 1

§2 / EXPECT期待値と分散

期待値 E[X] = Σ x · P(X=x) （離散） = ∫ x · f(x) dx （連続）
分散 Var[X] = E[(X - E[X])²] = E[X²] - E[X]²

期待値は 「無限回試行したときの平均」。分散はばらつき。標準偏差 σ = √Var。

§4 / CLT中心極限定理

機械学習で最も重要な統計定理。独立同分布な確率変数 X₁, ..., Xₙ の平均は、n が大きいとき正規分布に従う。元の分布が 何であっても 成り立つ。

FIG 5.2 中心極限定理：サイコロのような一様分布から取ったサンプルの平均でも、n を大きくすると正規分布に近づく。これが「信頼区間」「仮説検定」を可能にする魔法。

実用上の意味：標本平均は正規分布で近似できる → 信頼区間が計算できる → 仮説検定ができる、という連鎖が動きます。ニューラルネットの BatchNorm が機能する直観的根拠の一つでもあります。

§5 / BAYESベイズの定理

P(A | B) = P(B | A) · P(A) / P(B)

パラメータ θ とデータ D に置き換えると：

P(θ | D) ∝ P(D | θ) · P(θ)
事後 ∝ 尤度 × 事前
FIG 5.3 ベイズの定理の図解。事前分布（最初の信念）と尤度（データから得た証拠）を掛けると、事後分布（更新された信念）になる。これがベイズ的思考の核心。

L2正則化はガウス事前、L1はラプラス事前 を仮定した MAP 推定と等価、というように機械学習との接続も深い。

§6 / MLE最尤推定（Maximum Likelihood Estimation）

観測データ x₁, ..., xₙ が与えられたとき、それを 最も「もっともらしく」する パラメータ θ を選ぶ：

θ̂_MLE = argmax_θ Π P(xᵢ | θ) = argmax_θ Σ log P(xᵢ | θ)
例題：ベルヌーイ分布の最尤推定
コイン投げ。n 回中 k 回表が出た。表の確率 p を MLE で推定。
尤度: L(p) = pᵏ(1-p)ⁿ⁻ᵏ
対数尤度: log L(p) = k log p + (n-k) log(1-p)
微分してゼロ: d/dp log L = k/p - (n-k)/(1-p) = 0
→ p̂ = k/n   結論：MLE は単に「成功比率」。直感と一致！

6.1 最尤推定と機械学習の損失関数

MLE は 対数尤度の最大化 ＝ 負の対数尤度（NLL）の最小化。これが多くの損失関数の正体：

  • 正規分布の MLE → 二乗誤差（MSE）損失（線形回帰）
  • ベルヌーイの MLE → 二値クロスエントロピー（ロジスティック回帰、二値分類）
  • カテゴリカルの MLE → 多クラスクロスエントロピー（softmax分類）

§7 / MAPMAP推定とベイズの接続

MAP推定（Maximum A Posteriori）：事後分布 P(θ | D) を最大化する θ。

θ̂_MAP = argmax_θ P(θ | D) = argmax_θ [P(D | θ) · P(θ)]

事前分布が一様（無情報事前）なら、MAP = MLE。事前を入れることで 正則化 が自然に出てくる：

  • 事前 P(θ) ∝ exp(-||θ||²) （ガウス）→ L2 正則化
  • 事前 P(θ) ∝ exp(-||θ||₁) （ラプラス）→ L1 正則化

§8 / CI信頼区間と仮説検定

信頼区間：パラメータの推定値の不確かさを表す区間。95% 信頼区間とは、同じ実験を多数回行うと、95% の区間が真の値を含むという意味（真の値が95%の確率で入る、ではない！）。

仮説検定：帰無仮説 H₀ vs 対立仮説 H₁。p値 が小さいと H₀ を棄却。p値はH₀ の確率ではない（H₀ が真と仮定したときに、観測値以上の極端な結果が得られる確率）。

第1種の過誤H₀ が真なのに棄却（偽陽性）。確率 α
第2種の過誤H₀ が偽なのに棄却しない（偽陰性）。確率 β
検出力1 - β。H₁ が真のとき正しく棄却する確率
