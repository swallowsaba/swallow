§1 / SPLIT訓練・検証・テスト分割

機械学習の最も基本的な実験設計。3つに分割：

FIG 9.1 データ分割の基本。テストデータは 「最終評価で一度しか見ない」。何度も見て調整すると暗黙の過学習が起こる。
訓練（train）モデルのパラメータ（重み）を学習
検証（validation）ハイパーパラメータ調整、モデル選択、早期終了
テスト（test）最終的な汎化性能の評価。複数回見てはいけない
⚠ データリーク
テストデータの情報が訓練に漏れることを データリーク という。標準化・特徴量設計を分割前にやると平均・分散にテストデータが入ってしまう。正しくは：分割 → 訓練データで統計を計算 → 検証/テストにも同じ変換を適用。

§2 / CVK-fold 交差検証

データが少ない場合、訓練データを K 分割し、各 fold を検証用にして K 回学習・評価、平均する。

FIG 9.2 5-fold 交差検証。データを5分割し、各 fold を順番に検証用にする。データ少なくても汎化性能を頑健に推定できる。

代表的なバリエーション：

  • Stratified K-fold：各 fold でクラス比率を一定に
  • LOO（Leave-One-Out）：K = データ数。極小データ向け
  • Time Series Split：時系列データでは未来を訓練に使わない
  • Group K-fold：同じ患者/ユーザーが訓練と検証に分かれないように

§3 / OVERFIT過学習と未学習

FIG 9.3 未学習・適合・過学習。同じデータでも、モデルの複雑度によって挙動が変わる。過学習はノイズまで学習している状態。

§4 / BIAS-VARバイアス・バリアンス分解

汎化誤差 = バイアス² + バリアンス + 既約誤差。

用語意味原因対策
バイアス真の関数からの系統的ずれモデル単純すぎ（underfit）モデル容量↑、特徴量増
バリアンス訓練データのノイズへの過敏さモデル複雑すぎ（overfit）正則化、データ拡大、Dropout
既約誤差データ自体に含まれるノイズ避けられない－

§6 / METRICS評価指標

Accuracy(TP + TN) / 全件。クラス不均衡だと役立たず
Precision（適合率）TP / (TP + FP)。陽性と判定したうち本当に陽性
Recall（再現率・感度）TP / (TP + FN)。本当の陽性をどれだけ捕まえたか
F12 · P · R / (P + R)。PとRの調和平均
Specificity（特異度）TN / (TN + FP)。本当の陰性をどれだけ正しく陰性と
FPRFP / (FP + TN) = 1 − Specificity
例題：がん検診の評価
検査結果：TP=80, FN=20, FP=200, TN=9700
Accuracy = (80+9700)/10000 = 97.8%（一見良い）
Precision = 80/(80+200) = 28.6%
Recall = 80/(80+20) = 80%
医療では Recall（見落としを減らす）が重要。Accuracy は不均衡データで誤解を招く。

§7 / ROCROC 曲線と AUC

分類器のスコア閾値を 0 → 1 で動かしたときの (FPR, TPR) の軌跡が ROC 曲線。閾値非依存の評価。

FIG 9.5 ROC 曲線。閾値を動かすと (FPR, TPR) がプロットされる。曲線の下面積 (AUC) が大きいほど分類器が良い。完璧 = 1.0、ランダム = 0.5。

PR 曲線・PR-AUC：不均衡データ（陽性が稀）では ROC より PR (Precision-Recall) 曲線 の方が適切な評価。

§8 / REGUL正則化の手段

  • L1 / L2 / Elastic Net：重みのペナルティ（章8）
  • Early Stopping：検証誤差が悪化し始めたら学習停止
  • Dropout（DL）：学習時に各ユニットを確率 p で 0 に（章12）
  • Batch Normalization：各層の入力を正規化（暗黙の正則化）
  • データ拡張：画像なら回転・反転、テキストなら言い換え
  • アンサンブル：複数モデルの予測を平均
  • Label Smoothing：one-hot ラベルを少しソフト化

§9 / SUMMARY章のまとめ

1訓練・検証・テストの3分割。テストは1度だけ見る
2データリークの防止：分割 → 訓練で統計計算 → 全体に適用
3K-fold CV、Stratified、Time Series Split、Group K-fold の使い分け
4過学習：バリアンス大。未学習：バイアス大
5汎化誤差 = バイアス² + バリアンス + 既約誤差
6混同行列：TP/FN/FP/TN の4区分
7Precision・Recall・F1 はクラス不均衡で重要
8ROC-AUC：閾値非依存の総合指標
9不均衡データでは PR-AUC の方が適切
10正則化：L1/L2、Dropout、Early Stopping、Data Augmentation
