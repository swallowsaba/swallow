行列を、
「変換」として読む。

線形代数の核心は 「行列 = 線形変換」 という1つの視点に集約されます。これを獲得すれば、固有値もSVDも自然に意味づけられます。

§1 / VECTORSベクトル：方向と大きさ

ベクトル は「方向と大きさを持つ量」。機械学習では 特徴量を並べたデータ点 として使います。

x = (x₁, x₂, ..., xₙ)ᵀ ∈ ℝⁿ
FIG 3.1 2次元ベクトル v = (3, 2)。原点から点 (3, 2) への矢印。長さ ||v|| = √(3² + 2²) = √13。

1.1 ベクトル和とスカラー倍

和は 平行四辺形則。スカラー倍は方向を保ったまま長さを変えます（負ならば反転）。

§2 / DOT内積：類似度の計算

内積 は機械学習の至るところで現れる。直感的には 「2つのベクトルがどれだけ同じ方向か」。

代数的: u · v = uᵀv = u₁v₁ + u₂v₂ + ... + uₙvₙ
幾何的: u · v = ||u|| · ||v|| · cos θ
FIG 3.2 内積の幾何学的意味。これが Word2Vec などの「埋め込み類似度」の基盤。

2.1 コサイン類似度

cos_sim(u, v) = (u · v) / (||u|| · ||v||) ∈ [-1, 1]

長さに依らず方向の近さだけを測る。埋め込みベクトル（Word2Vec、Sentence-BERT、CLIP） の類似度計算で多用。

例題：単語埋め込みの類似度
v_東京 = (0.4, 0.1, 0.7), v_大阪 = (0.5, 0.0, 0.6)
内積 = 0.4×0.5 + 0.1×0 + 0.7×0.6 = 0.62
||v_東京|| ≈ 0.812, ||v_大阪|| ≈ 0.781
cos_sim ≈ 0.977 → 非常に類似（「都市」概念の近さ）

§3 / NORMノルム：長さの一般化

ノルム定義幾何用途
L¹（マンハッタン）Σ|xᵢ|菱形の単位球Lasso、スパース化
L²（ユークリッド）√Σxᵢ²円の単位球Ridge、距離
L∞max|xᵢ|正方形の単位球敵対的攻撃の制約
FIG 3.3 3つのノルムの単位球（ノルムが1のベクトルの集合）。Lasso が L¹ を使うと菱形の角に解が来やすく、スパース化（多くの重みが 0）を引き起こす。

§4 / MATRIX行列：線形変換

本章の核心：m×n 行列 A は、n 次元ベクトルを m 次元ベクトルに変換する関数。

A : ℝⁿ → ℝᵐ,   y = Ax

4.1 2次元の典型変換

FIG 3.4 2次元の典型的な線形変換。点線が変換前、実線が変換後。行列を「変換の指示書」として読むのが本質。

§5 / PRODUCT行列の積

行列の積 AB は 「変換 B の後に変換 A を適用」する合成変換（書く順と適用順が逆）。

非可換性：AB ≠ BA（一般に）。

Cᵢⱼ = Σₖ Aᵢₖ · Bₖⱼ
例題：2×2 行列の積
A = [[1, 2], [3, 4]], B = [[5, 6], [7, 8]]
C₁₁ = 1×5 + 2×7 = 19,   C₁₂ = 1×6 + 2×8 = 22
C₂₁ = 3×5 + 4×7 = 43,   C₂₂ = 3×6 + 4×8 = 50
C = AB = [[19, 22], [43, 50]]

§6 / DET行列式と逆行列

行列式 det(A) は「変換 A による面積（体積）の倍率」。

2×2: det([[a,b],[c,d]]) = ad - bc
  • det(A) = 0 → 次元を潰す変換 → 非可逆
  • det(A) > 0 → 向きを保つ変換
  • det(A) < 0 → 向きが反転（鏡映を含む）
  • det(AB) = det(A)·det(B)

逆行列 A⁻¹：A の変換を逆向きに戻す変換。det(A) ≠ 0 のとき存在。

連立方程式 Ax = b の解：A が正則ならば x = A⁻¹b。
機械学習の 線形回帰の正規方程式 ŵ = (XᵀX)⁻¹Xᵀy はこの応用。XᵀX が非可逆（多重共線性）だと不安定 → リッジ回帰で逃げる。

§7 / EIGEN固有値と固有ベクトル

行列 A に対して、ベクトル v ≠ 0 とスカラー λ が Av = λv を満たすとき、v を 固有ベクトル、λ を 固有値。

意味：「行列 A をかけても方向が変わらず、大きさだけ λ 倍になる特別な方向」。

FIG 3.5 固有ベクトルだけは、行列 A による変換で「方向」が保たれ、長さだけが λ 倍になる。これが固有値の本質。

7.1 固有値方程式の導出

Av = λv より (A - λI)v = 0。v ≠ 0 の解を持つには (A - λI) が非可逆、すなわち：

det(A - λI) = 0   (固有方程式)

これは λ に関する n 次方程式。n 個の固有値（複素・重複含む）が得られる。

例題：2×2 行列の固有値
A = [[3, 1], [0, 2]] の固有値を求める。
det(A - λI) = (3-λ)(2-λ) - 0 = 0 → λ = 3, 2
λ = 3 の固有ベクトル：(A - 3I)v = 0 → [[0,1],[0,-1]]v = 0 → v = (1, 0)ᵀ
λ = 2 の固有ベクトル：(A - 2I)v = 0 → [[1,1],[0,0]]v = 0 → v = (1, -1)ᵀ
→ x 軸方向は 3 倍、(1, -1) 方向は 2 倍に伸縮される変換

7.2 機械学習での活用

  • PCA：データの共分散行列の固有ベクトル = 最大分散方向（章8）
  • ヘッセ行列の固有値：最適化での局所形状判定（章4 と 章11）
  • Google PageRank：リンク行列の最大固有ベクトル

§8 / SVD特異値分解（SVD）

任意の m×n 行列 A を3つの行列の積に分解：

A = U Σ Vᵀ
  • U: m×m 直交行列（左特異ベクトル）
  • Σ: m×n 対角様行列（特異値 σ₁ ≥ σ₂ ≥ ... ≥ 0）
  • V: n×n 直交行列（右特異ベクトル）
FIG 3.6 SVD は任意の線形変換を「回転 → スケーリング → 回転」の3ステップに分解する。特異値 σᵢ は各軸の伸び率を表す。

8.1 低ランク近似

上位 k 個の特異値だけを残せば、元の行列を低ランクで近似できる：

A ≈ U_k Σ_k V_kᵀ   (rank k)

これが データ圧縮・画像圧縮・推薦システム（Netflix Prize）・LoRA（LLMの効率的微調整）の基礎です。

§10 / SUMMARY章のまとめ

1ベクトル：方向と大きさ。機械学習では特徴量を並べたデータ点
2内積：類似度の数学。コサイン類似度は方向の近さ
3L¹/L²/L∞ ノルム：それぞれ異なる「長さ」の定義
4行列 = 線形変換。これが本章で最も重要な視点
5行列の積 = 変換の合成（非可換）
6行列式 = 面積倍率。0 なら非可逆
7固有値・固有ベクトル：変換で方向が保たれる特別な軸
8SVD = 任意の行列を「回転・スケール・回転」に分解
9低ランク近似がデータ圧縮・LoRA の基礎
10PCA、Lasso、線形回帰、Attention、LoRA... 全て線形代数
