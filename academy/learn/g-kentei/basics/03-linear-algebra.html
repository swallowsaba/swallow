G-KENTEI · BASICS · CHAPTER 3

機械学習は、
行列の言葉で書かれている。

『深層学習を学ぶには線形代数が必要だ』とよく言われます。なぜでしょうか。ニューラルネットワークの順伝播 y = σ(Wx + b) は、本質的に 行列ベクトル積 です。畳み込みは行列の特殊な形で書け、Attention の softmax(QKT/√d)V は3つの行列積の連鎖です。PCA は 共分散行列の固有値分解、画像圧縮は SVD のランク削減、グラフ解析は 隣接行列の固有値 ── 機械学習のあらゆる計算は、線形代数の言葉で記述されています。本章で、それを母語にしましょう。

SECTION ONE

基本

テンソルの階層、ベクトル・行列演算、ノルム、線形独立、階数、行列式、逆行列、固有値分解、SVD、二次形式の10節。

基本

§1.1 スカラーからテンソルへ — データの次元の階層

機械学習で扱う数値オブジェクトは、その『次元（軸）の数』で4段階に分類されます。これがすべての基本です。

名称軸の数表記例
スカラー (scalar)0軸（1個の数）x ∈ ℝ気温 25.3、学習率 0.001
ベクトル (vector)1軸（n個の数の列）v ∈ ℝn3次元位置 (x, y, z)、Word Embedding 768次元
行列 (matrix)2軸（縦×横）A ∈ ℝm×nグレースケール画像（縦×横）、重み行列 W
テンソル (tensor)3軸以上T ∈ ℝd₁×d₂×…×d_kカラー画像（高さ×幅×3チャネル）、動画（時間×高さ×幅×チャネル）
FIG 3.1 ／ スカラー → ベクトル → 行列 → テンソルの階層。深層学習で扱うデータはすべてこの4段階で表現される。カラー画像は (H, W, 3) の3階テンソル、動画は (T, H, W, 3) の4階テンソル。

NumPy では shape 属性でこの階層を確認できます：

np.array(5).shape → () ← スカラー
np.array([1,2,3]).shape → (3,) ← ベクトル
np.array([[1,2],[3,4]]).shape → (2,2) ← 行列
np.zeros((3,224,224)).shape → (3,224,224) ← テンソル（RGB画像）
▎ 用語の注意
物理学の『テンソル』は座標変換則を満たす特別な対象を指しますが、機械学習では 単に多次元配列 という意味で使われます。TensorFlow の『Tensor』も後者の意味です。

§1.2 ベクトル演算 — 加算・スカラー倍・内積

ベクトルに対する基本演算は3つ：加算、スカラー倍、内積。前2つは要素ごとの計算で済みますが、内積こそが機械学習の中核演算です。

加算　 u + v = (u₁+v₁, u₂+v₂, ..., u_n+v_n)
スカラー倍 αv = (αv₁, αv₂, ..., αv_n)
内積（ドット積）　 u · v = Σ u_i v_i = uTv = ||u|| ||v|| cosθ

内積の幾何学的意味は『射影の長さ × ベクトルの長さ』。これが類似度の自然な指標になり、コサイン類似度・Attention・カーネル法など、機械学習の至るところで現れます。

FIG 3.2 ／ 内積の幾何学的意味。要素積の和、||u||||v||cosθ、射影 × 長さ の3つの解釈はすべて等価。直交性の判定はAttention・PCA・直交初期化など至るところで使われる。

内積の応用は多岐にわたります：

  • コサイン類似度： cos(u,v) = u·v / (||u|| ||v||) ── 単語埋め込み類似度の標準
  • Attention 重み： softmax(Q·KT/√d) ── Transformer の核
  • ニューロン活性： z = w·x + b ── 1ニューロンの計算は重みベクトルと入力の内積
  • カーネル法： K(x,x') = φ(x)·φ(x') ── SVM 等の非線形分離
  • 正規方程式： XTX w = XTy ── 線形回帰の解

§1.3 行列演算 — 積・転置・トレース

行列の最重要演算は 行列積。「左の行 × 右の列の内積」というルールで、A∈ℝm×k と B∈ℝk×n の積は AB∈ℝm×n。内側の次元 k が一致する必要があり、これが NN 設計時の最大の注意点です。

(AB)ij = Σk Aik Bkj　　　 ※ 左i行と右j列の内積

計算量： O(mkn)　　　例：1000×1000 × 1000×1000 = 10億回の積和（GPU で 1ms 未満）

注意： AB ≠ BA（一般に非可換）　 (AB)T = BTAT（転置の積は逆順）
FIG 3.3 ／ 行列積の計算ルール。左の i 行と右の j 列の内積が結果の (i,j) 要素になる。NN 設計でレイヤー間の次元不一致エラーは、この『内側の次元』の不一致がほぼ 100%。

転置 AT：行と列を入れ替える。(AT)ij = Aji。重要な性質：(AB)T = BTAT（順序が逆）。

トレース tr(A)：対角成分の和。 tr(A) = Σ Aii。性質：tr(AB) = tr(BA)（巡回不変性）。

アダマール積 A ⊙ B：要素ごとの積。形が同じである必要がある。NN の Dropout や Element-wise 演算で多用。

▎ NNでの行列積
隠れ層の計算 h = σ(Wx + b) で、入力 x∈ℝ784（MNISTの28×28画像）、重み W∈ℝ128×784、バイアス b∈ℝ128 なら、行列積 Wx は (128, 784) × (784,) = (128,) というベクトルを生成。これに b を足して非線形関数 σ（ReLU等）を通すと、128次元の隠れ層出力 h が得られる。ニューラルネットの順伝播は、本質的に行列ベクトル積の連鎖。

§1.4 ノルム — ベクトルの『長さ』を測る

ノルム ||v|| は『ベクトルの長さ』の一般化。文脈に応じて複数の定義があり、機械学習ではどれを使うかで結果が大きく変わります。

ノルム定義性質・用途
L1（マンハッタン）||v||₁ = Σ |vᵢ|角点に解 → スパース化。Lasso 正則化、特徴選択
L2（ユークリッド）||v||₂ = √(Σ vᵢ²)標準的『長さ』。Ridge 正則化、最も自然な距離
L∞（最大）||v||∞ = maxᵢ |vᵢ|最大成分。敵対的攻撃の上限制約
Lp（一般）||v||_p = (Σ |vᵢ|p)1/pp=1,2,∞ を含む一般化
Frobenius||A||_F = √(Σᵢⱼ Aᵢⱼ²)行列のL2版。重み正則化、低ランク近似
FIG 3.4 ／ L1・L2・L∞ ノルムの『単位球』。角の有無が正則化の効果を決める。L1の角が軸上にあることで、最適化解がしばしば軸上＝スパース解になる（Lassoの特徴選択効果）。
→ 関連
正則化での L1・L2 の選択は、章8（古典的機械学習）の Ridge・Lasso、章12（NNの工夫）の重み減衰、章9（評価）の頑健性で繰り返し登場します。

§1.5 線形独立と基底 — 情報の重複を測る

ベクトル v₁, v₂, …, v_k が 線形独立 とは、α₁v₁ + α₂v₂ + … + α_kv_k = 0 を満たす係数が α_i = 0 のみであること。直観的には『どのベクトルも他の組み合わせで作れない』状態。逆に 線形従属 なら、少なくとも1つのベクトルが冗長。

基底（basis）　 空間 V を張る線形独立なベクトル集合 {e₁, e₂, …, e_n}
　　　　　　　 V のどんなベクトルも基底の一意な線形結合で表せる

正規直交基底　 すべて単位ベクトル（||eᵢ||=1）かつ互いに直交（eᵢ·eⱼ=0, i≠j）
　　　　　　　 例：標準基底 e₁=(1,0,0), e₂=(0,1,0), e₃=(0,0,1)

次元（dimension）　 基底に含まれるベクトルの数（基底の選び方によらず一定）
FIG 3.5 ／ 線形独立は『どのベクトルも他の組み合わせで作れない』状態。線形従属だと情報が重複し、回帰の多重共線性、PCAの低ランク特異性として顕在化する。

§1.6 階数（rank）と零空間 — 行列が含む情報量

行列 A の 階数 rank(A) は、A の列ベクトル（または行ベクトル）のうち線形独立なものの最大個数。これが 『その行列が含む独立な情報の数』 を表します。

フルランク　 rank(A) = min(m, n) のとき、A は最大の情報量を持つ
ランク欠損　 rank(A) < min(m, n) のとき、情報重複あり

零空間（null space）　 Av = 0 を満たすすべての v の集合
　　　　　　　　　　 dim(零空間) + rank(A) = 列数 n　（次元定理）

機械学習での意味：

  • データ行列 X (n_sample × n_feature) の rank ── 真の情報量。特徴量間に強い相関があるとランク欠損。
  • 低ランク近似 ── 大きな行列を小さなランクで近似（SVD で実現）。LoRA, レコメンド, 圧縮の原理。
  • 逆行列の存在条件 ── A が n×n でフルランク（rank = n）のとき、かつそのときに限り A⁻¹ が存在。

§1.7 行列式と逆行列 — 線形写像の体積変化と反転

行列式 det(A) は正方行列 A に対して定義される1つの数値で、幾何学的には『A が表す線形写像による体積の倍率』を意味します。

2×2 行列の行列式　 det([[a,b],[c,d]]) = ad - bc

性質
　 det(AB) = det(A) det(B)
　 det(AT) = det(A)
　 det(A⁻¹) = 1 / det(A)
　 det(A) = 0 ⇔ A は特異（逆行列なし、ランク欠損）

幾何的意味
　 |det(A)| = 写像による体積の倍率
　 sign(det(A)) = 向きの反転（負なら鏡映を含む）

逆行列 A⁻¹：AA⁻¹ = A⁻¹A = I を満たす行列。存在条件は det(A) ≠ 0（= フルランク）。線形方程式 Ax = b の解は x = A⁻¹b。ただし実用上、逆行列を陽に計算するより LU分解・QR分解・SVD で解く方が数値的に安定。

▎ よくある罠
『逆行列があるかどうか』を det(A) で判定するのは数学的には正しいが、数値計算では浮動小数点誤差で『ほぼ 0』の det が出る場合が多い。実用では 条件数（condition number） κ(A) = σ_max / σ_min（最大／最小特異値の比）で『近似的特異性』を判定する。κ > 10⁶ で警戒、> 10¹⁰ で計算結果はほぼ信用できない。

§1.8 固有値分解 — 線形写像の『主軸』を見つける

正方行列 A に対し、Av = λv を満たす非零ベクトル v を 固有ベクトル、対応するスカラー λ を 固有値 と呼びます。意味は『A による変換で 方向が変わらず、ただ λ 倍に伸縮するだけ』のベクトル。

固有値方程式　 Av = λv　 ⇔ 　(A - λI)v = 0
　　　　　　　 非自明解の条件： det(A - λI) = 0 ←特性方程式

固有値分解　 A = QΛQ⁻¹　 (Q：固有ベクトル列、Λ：固有値対角行列)
　　　　　　 適用条件：A が n 個の線形独立な固有ベクトルを持つ（対角化可能）

対称行列の場合　 A = QΛQT　 (Q は直交行列、Λ は実対角)
　　　　　　　　　 すべての対称行列は直交対角化可能（実スペクトル定理）
FIG 3.6 ／ 固有値分解の幾何学的意味。円が楕円に変換され、固有ベクトル方向は 方向を保ち、固有値の倍率で 伸縮 する。固有値分解とは『この主軸と倍率を見つける』こと。

固有値分解の応用：

  • PCA ── 共分散行列の固有値分解で主成分を抽出
  • Spectral Clustering ── グラフラプラシアンの固有ベクトルでクラスタリング
  • PageRank ── Web リンク行列の主固有ベクトル
  • マルコフ連鎖の定常分布 ── 遷移行列の固有値1の固有ベクトル
  • NNの最適化解析 ── ヘッセ行列の固有値で曲率（鞍点、極小、平坦性）を診断

§1.9 特異値分解（SVD） — 線形代数の万能ツール

固有値分解は正方行列にしか使えませんが、SVD は任意の m×n 行列に適用可能な拡張版です。これが機械学習の最重要分解。

SVD　 A = UΣVT　　　 (A ∈ ℝm×n)
　 U ∈ ℝm×m　 直交行列（左特異ベクトルの列）
　 Σ ∈ ℝm×n　 対角行列（特異値 σ₁ ≥ σ₂ ≥ … ≥ 0）
　 VT ∈ ℝn×n　 直交行列の転置（右特異ベクトルの行）

関係　 ATA の固有値 = σᵢ²　 ⇒ 特異値は ATA の固有値の平方根
ランク　 rank(A) = 非零の特異値の個数
FIG 3.7 ／ SVDの分解。A = UΣVT。U と V は直交行列、Σ は対角に非負の特異値を降順で並べた行列。任意の長方形行列に対して必ず存在する。

SVD の応用は無数にあります：

  • 低ランク近似（Eckart-Young 定理）：上位 k 個の特異値だけ残すと、最良の rank-k 近似。画像圧縮、ノイズ除去。
  • PCA の実装：共分散行列を陽に作らず SVD で直接（数値的に安定）。
  • Moore-Penrose 擬似逆行列：A+ = VΣ+UT。長方形行列の『逆行列の自然な拡張』。
  • Latent Semantic Indexing（LSI）：単語-文書行列の SVD で潜在意味を抽出。
  • レコメンドシステム：ユーザー-アイテム評価行列の SVD（Netflix Prize で有名）。
  • LoRA：LLM 微調整での低ランク差分行列の理論的根拠。

§1.10 二次形式と正定値性 — 凸最適化の根幹

対称行列 A による 二次形式： f(x) = xTAx = Σᵢⱼ Aᵢⱼ xᵢxⱼ。この符号によって関数の凸性が決まります。

分類条件意味
正定値 (positive definite, PD)全 x ≠ 0 で xTAx > 0全固有値 > 0、関数は凸、唯一の極小
半正定値 (PSD)全 x で xTAx ≥ 0全固有値 ≥ 0、共分散行列の必要条件
不定値 (indefinite)正と負の値をとる固有値に正負が混在、鞍点を含む
半負定値 / 負定値≤ 0 / < 0全固有値 ≤ 0 / < 0、関数は凹

ヘッセ行列（2階微分行列）の正定値性は、最適化問題における極小・極大・鞍点の判定に直結します（章4で詳述）。

応用

§2.1 ニューラルネットの行列演算 — 1行で書けるNN

3層 NN の順伝播は、本質的に行列積3回＋非線形変換2回です。

入力　 x ∈ ℝ784　 （28×28画像をflat化）
1層目　 h₁ = ReLU(W₁ x + b₁)　 W₁ ∈ ℝ128×784, b₁ ∈ ℝ128
2層目　 h₂ = ReLU(W₂ h₁ + b₂)　 W₂ ∈ ℝ64×128, b₂ ∈ ℝ64
出力　 y = softmax(W₃ h₂ + b₃)　 W₃ ∈ ℝ10×64, b₃ ∈ ℝ10

パラメータ総数 = 784·128 + 128 + 128·64 + 64 + 64·10 + 10 = 約 109,000 個。これでも MNIST で 97% 程度の精度を出せます。すべて行列とベクトルの言葉で書け、GPU はこの行列積を超高速で並列実行します。

FIG 3.8 ／ 3層 NN は本質的に行列ベクトル積3回＋非線形変換。すべての深層学習はこの構造の拡張に過ぎない。GPU の高速行列積（Tensor Core）が深層学習を可能にした基盤技術。

§2.2 PCA — 共分散行列の固有値分解

PCA（主成分分析）は 高次元データを分散最大方向に射影 する次元削減法。手順は：

1. データを中心化：X ← X - mean(X)
2. 共分散行列：Σ = (1/n) XTX　（n × n から d × d に縮約）
3. 固有値分解：Σ = QΛQT　（Λ の対角が各方向の分散）
4. 上位 k 個の固有ベクトル（主成分）に射影：Y = X Q[:, :k]

第 i 主成分の 寄与率 = λᵢ / Σλⱼ（その方向が説明する分散の割合）。累積寄与率 90% などを基準に k を選ぶ。実装は SVD で行うのが数値的に安定：X = UΣVT なら主成分は V の列。

PCA の応用：(a) 可視化（高次元 → 2D/3D）、(b) ノイズ除去（低ランク化）、(c) 特徴抽出の前処理、(d) 圧縮、(e) 推薦システムの初期化、(f) Whitening（白色化）── 各方向の分散を1に揃える前処理で、勾配の偏りを除去。

§2.3 共分散行列とマハラノビス距離

共分散行列 Σ：Σᵢⱼ = Cov(Xᵢ, Xⱼ)。対角がそれぞれの分散、非対角が変数間の共分散。性質：常に半正定値、対称。確率密度関数（多変量正規分布）、相関、PCA、線形判別、ガウス過程など、機械学習の至るところで現れる中核オブジェクト。

多変量正規分布
　 p(x) = (2π)-d/2 |Σ|-1/2 exp(-½ (x-μ)TΣ⁻¹(x-μ))

マハラノビス距離　 d_M(x, μ) = √((x-μ)TΣ⁻¹(x-μ))
　 分布の形状（共分散）を考慮した距離。外れ値検出、異常検知で標準。
　 Σ = I なら通常のユークリッド距離に一致。

§2.4 Self-Attention の Q·K·V 構造

Transformer の核 Self-Attention は3つの行列演算の連鎖です。これが現代 LLM の基盤。

入力　 X ∈ ℝL×d　 (L=系列長, d=モデル次元)

Q, K, V を生成　 Q = XWQ, K = XWK, V = XWV　 (W は学習可能パラメータ)

類似度　 S = QKT / √dk　 (L × L 行列)

Attention 重み　 A = softmax(S)　 (各行が確率分布)

出力　 Y = AV　 (L × d 行列、各位置が全位置の重み付き和)
FIG 3.9 ／ Self-Attention は3つの行列積（XW, QKT, AV）と1回の softmax。GPT・BERT・Claude・Geminiなど、すべての現代 LLM の本体はこの構造を多層に積み重ねたもの。

計算量：O(L²d)。系列長 L が長いと L² が支配的になり、超長文（数万トークン）で問題に。Flash Attention（IO 効率化）、Mamba/SSM（O(L) 計算量の代替）など、線形代数の工夫が現在も活発に研究されています。

§3.3 固有値分解の実装

対称行列の固有値分解を NumPy で行い、A = QΛQT を確認します。

# 対称行列を作る np.random.seed(42) M = np.random.randn(4, 4) A = (M + M.T) / 2 # 対称化 # 固有値分解 (対称行列用：eigh は実固有値を保証) eigenvalues, eigenvectors = np.linalg.eigh(A) print("固有値:", eigenvalues) print("固有ベクトル (列ベクトルとして):") print(eigenvectors) # 再構成 A = Q Λ Q^T を検算 Lambda = np.diag(eigenvalues) A_reconstructed = eigenvectors @ Lambda @ eigenvectors.T print("再構成誤差:", np.linalg.norm(A - A_reconstructed)) # 浮動小数点誤差程度の値（10^-15 オーダー） # 直交性の確認 Q^T Q = I print("Q^T Q =") print(np.round(eigenvectors.T @ eigenvectors, 10))

§3.4 SVD による画像圧縮

SVD の低ランク近似で、画像を少数の特異値だけで再構成します。Eckart-Young 定理により、これが最良の rank-k 近似。

from sklearn.datasets import fetch_olivetti_faces img = fetch_olivetti_faces().images[0] # 64×64 顔画像 # SVD U, sigma, Vt = np.linalg.svd(img, full_matrices=False) print("画像サイズ:", img.shape) print("特異値の総数:", len(sigma)) print("特異値の最大10個:", sigma[:10]) # ランクk近似 def low_rank_approx(U, sigma, Vt, k): return U[:, :k] @ np.diag(sigma[:k]) @ Vt[:k, :] for k in [1, 5, 10, 20, 50]: approx = low_rank_approx(U, sigma, Vt, k) error = np.linalg.norm(img - approx, 'fro') compression = (k * (64 + 64 + 1)) / (64 * 64) * 100 print(f"k={k:3d}: 誤差={error:.4f}, 圧縮率={compression:.1f}%")
OUTPUT

§3.5 PCAをスクラッチ実装

PCA を NumPy だけで実装し、sklearn の結果と一致することを確認します。

from sklearn.datasets import load_iris from sklearn.decomposition import PCA as SklearnPCA iris = load_iris() X = iris.data # 150 サンプル × 4 特徴量 # スクラッチ実装 def pca_from_scratch(X, n_components): X_centered = X - X.mean(axis=0) # 中心化 cov = X_centered.T @ X_centered / (len(X) - 1) # 共分散行列 eigenvalues, eigenvectors = np.linalg.eigh(cov) # 固有値降順に並べる idx = np.argsort(eigenvalues)[::-1] eigenvalues = eigenvalues[idx] eigenvectors = eigenvectors[:, idx] # 上位 k 個 components = eigenvectors[:, :n_components] X_reduced = X_centered @ components return X_reduced, eigenvalues[:n_components], components X_my, my_var, my_comp = pca_from_scratch(X, n_components=2) # sklearn と比較 sk = SklearnPCA(n_components=2) X_sk = sk.fit_transform(X) # 寄与率 total_var = X.var(axis=0, ddof=1).sum() print("自作 寄与率:", my_var / total_var) print("sklearn: ", sk.explained_variance_ratio_) # [0.9246, 0.0531] — 第1主成分だけで 92% を説明

SIM 1 — 高次元での「距離の呪い」

d 次元の単位立方体に一様に分布した点群を考える。次元 d を増やすと、点同士の距離分布はどう変化するか？
>> 各次元 d で、1000点をランダム生成、全ペア距離を計算

d=2: min距離=0.012, max距離=1.391, max/min=119.1
d=10: min距離=0.341, max距離=2.156, max/min=6.32
d=100: min距離=2.871, max距離=4.892, max/min=1.70
d=1000: min距離=11.84, max距離=13.97, max/min=1.18

>> 高次元では『すべての点が遠く、ほぼ等距離』に近づく
>> k-NN や K-means が高次元で機能しない理由
>> PCA・Embedding で次元削減する動機

この現象は 距離の呪い（curse of dimensionality） として知られています。深層学習が高次元データ（画像 50万次元、テキスト 768次元）で機能するのは、データが実は低次元の 多様体 に存在しているからです。線形代数の SVD・PCA、深層学習の Autoencoder はすべてこの『データの真の次元』を見つけようとする試み。

>> 全特異値の和（ノルム）と上位 k 個の寄与

k=5: 累積寄与率= 67.4%
k=10: 累積寄与率= 89.2%
k=20: 累積寄与率= 98.1%
k=50: 累積寄与率= 99.9%

>> 一方、自然画像の SVD では：
k=1: 累積寄与率= 78.3%　 ← はるかに集中
k=10: 累積寄与率= 98.4%
>> 構造を持つデータほど低ランクで近似可能

SIM 3 — テンソル形状の追跡

CNN の典型的なテンソル形状の変化を追跡。バッチサイズ 32 の ImageNet 画像が ResNet-18 を通過するとき。
>> 入力:　　　　 (32, 3, 224, 224)　 = 4階テンソル
>> Conv1+Pool: (32, 64, 56, 56)
>> Layer1:　　 (32, 64, 56, 56)

>> Layer3:　　 (32, 256, 14, 14)

>> Flatten:　　 (32, 512)　　　　　 = 2階（ベクトルのバッチ）
>> FC:　　　　 (32, 1000)　　　　 = ImageNet 1000クラス

>> 全要素数：32×3×224×224 ≈ 480万 → 32×1000 = 3.2万
>> ネット全体は情報を集約しつつ抽象化していく

G検定形式の演習問題 20問。線形代数の各概念を、実際の試験レベルで確認する。

問題

PROBLEM 1
機械学習で扱う『テンソル』の最も適切な説明は次のうちどれか。
① 物理学の座標変換則を満たす対象
② スカラーまたはベクトルのみ
③ 軸が3つ以上のものに限る
④ スカラー(0軸)・ベクトル(1軸)・行列(2軸)・3軸以上 を含む多次元配列の総称
ANSWER ④ 機械学習における『テンソル』は単に多次元配列の意味で使われ、軸数を問わない。TensorFlow の Tensor も同様。物理学の意味は別。
PROBLEM 2
RGB カラー画像 (高さ 224, 幅 224, チャネル 3) は何階テンソルか。
① 1階　 ② 2階　 ③ 3階　 ④ 4階
ANSWER ③ 軸が3つ（H, W, C）あるため3階テンソル。これにバッチ次元を加えると (B, H, W, C) または (B, C, H, W) の4階テンソルとなり、CNN の入力形式となる。
PROBLEM 3
ベクトル u=(2, 3, 1) と v=(1, -1, 2) の内積 u·v は。
① 0　 ② 1　 ③ 3　 ④ 7
ANSWER ② 2·1 + 3·(-1) + 1·2 = 2 - 3 + 2 = 1。
PROBLEM 4
2つのベクトルが直交するとき、内積の値は。
① 必ず正　 ② 必ず負　 ③ 0　 ④ 1
ANSWER ③ u·v = ||u||||v||cosθ で、直交（θ=90°）なら cos90°=0 のため 0。これが Word Embedding の独立性、Attention の出力の独立次元、直交初期化の根拠。
PROBLEM 5
行列 A∈ℝ3×4 と B∈ℝ4×2 の積 AB のサイズは。
① 3×2　 ② 4×4　 ③ 3×4　 ④ 計算不可
ANSWER ① 行列積は (m×k) × (k×n) → (m×n)。内側の次元 k=4 が一致するので計算可能、結果は外側の次元 3×2。
PROBLEM 6
L1 ノルムが Lasso 正則化で『スパース解』を生む理由として正しいのは。
① L1単位球が円形で滑らかなため
② L1単位球が菱形で『角』があり、最適解が軸上に来やすいため
③ L1ノルムが微分可能なため
④ L1ノルムが指数関数を含むため
ANSWER ② L1単位球（||w||₁=1）は菱形で、その『角』は軸上にある。制約付き最適化の解は等高線と単位球が接する点であり、角に当たることが多く、その軸の重みが正確に 0 になる（スパース化）。L2 の円には角がなく、すべての重みが小さく縮むだけ。
PROBLEM 7
ベクトル v₁=(1, 2)、v₂=(2, 4) が形作る関係として正しいのは。
① 線形独立で平面を張る
② 直交している
③ 線形従属（v₂ = 2 v₁）で直線のみを張る
④ ノルムが等しい
ANSWER ③ v₂ = 2 v₁ なので一方が他方の定数倍。すなわち線形従属で、両者は同じ直線上にあり、平面を張れない。
PROBLEM 8
正方行列 A の逆行列が存在する必要十分条件は。
① det(A) = 0　 ② det(A) ≠ 0 かつ A はフルランク　 ③ A が対称　 ④ A の全要素が正
ANSWER ② det(A) ≠ 0 と『フルランク（rank=n）』は同値。これらが逆行列存在の条件。実用では条件数 κ(A) で『近似的特異性』も警戒する。
PROBLEM 9
固有値分解 A = QΛQ⁻¹ が常に可能となる行列の特徴は。
① 全ての正方行列
② 対称（実対称）行列 — このとき Q は直交行列、Λ は実対角
③ 特異行列のみ
④ 上三角行列のみ
ANSWER ② 一般の正方行列は固有値分解が常に可能とは限らない（Jordan 標準形が必要なケースあり）。しかし対称行列は実スペクトル定理により、必ず実固有値で直交対角化可能。PCA・カーネル法・ヘッセ行列の解析がうまく行く根拠。
PROBLEM 10
特異値分解（SVD）A = UΣVT に関する記述で正しいのは。
① 正方行列にしか適用できない
② 任意の実行列に適用可能、Σ は非負の特異値を降順に対角に持つ
③ A の固有値とΣの対角は等しい
④ U と V は同じ行列
ANSWER ② SVDは固有値分解の自然な一般化で、任意の m×n 実行列に適用可能。U と V は直交、Σ は非負の特異値を降順に並べた対角行列。特異値は ATA の固有値の平方根。
PROBLEM 11
PCA の理論的根拠として最も適切な記述は。
① 共分散行列の固有値分解で分散最大方向（主成分）を見つける
② クラスラベルの分散を最大化
③ データを log 変換する
④ 重みを正則化する
ANSWER ① PCA は中心化したデータの共分散行列 Σ = (1/n)XTX を固有値分解し、固有値が大きい順の固有ベクトル（主成分）にデータを射影する。教師ラベルは使わない（教師なし）。クラス分離は LDA。
PROBLEM 12
対称行列 A の固有値がすべて正のとき、A は何と呼ばれるか。
① 半正定値　 ② 正定値（positive definite）　 ③ 不定値　 ④ 対角行列
ANSWER ② 対称行列で全固有値 > 0 のとき正定値（PD）。等号許可なら半正定値（PSD、全固有値 ≥ 0）。共分散行列は必ず PSD、最適化のヘッセ行列が PD ならその点は極小。
PROBLEM 13
Self-Attention の計算 softmax(QKT/√d)V の計算量は系列長 L に対して。
① O(L)　 ② O(L log L)　 ③ O(L²)　 ④ O(L³)
ANSWER ③ QKT が L×L の類似度行列を作るため O(L²)。長系列で支配的なボトルネック。Flash Attention、Mamba/SSM、Linformer などの研究はこの計算量を削減する試み。
PROBLEM 14
行列 A、B について (AB)T はどう書けるか。
① ATBT　 ② BTAT　 ③ AB　 ④ ATB
ANSWER ② 転置は積の順序を逆にする：(AB)T = BTAT。逆行列も同様：(AB)⁻¹ = B⁻¹A⁻¹。NN の逆伝播計算でこの公式は頻繁に登場。
PROBLEM 15
行列式 det(A) の幾何学的意味として正しいのは。
① A の各要素の平均
② A が表す線形写像による体積の倍率
③ A の固有値の数
④ A の行数
ANSWER ② |det(A)| は A による線形写像が単位立方体に作用したときの体積倍率。負なら向きが反転（鏡映）。det = 0 は『写像が次元を潰す』ことを意味し、逆写像が不可能。
PROBLEM 16
機械学習で『多重共線性』が問題となる理由として最も適切なのは。
① 計算速度が遅くなる
② 特徴量行列のランクが落ち、(XTX)⁻¹ が不安定になり係数推定が安定しない
③ 損失関数が複雑になる
④ データが増えすぎる
ANSWER ② 線形回帰の正規方程式 w = (XTX)⁻¹XTy は XTX の可逆性に依存。特徴量間に強い線形関係があると XTX はほぼ特異になり、係数推定の標準誤差が爆発、解釈不能に。対処は VIF 検出、変数削減、PCA、Ridge 回帰など。
PROBLEM 17
Eckart-Young 定理が示すことは。
① 任意の行列は LU 分解できる
② 行列の最良の rank-k 近似は、上位 k 個の特異値を残した SVD で得られる
③ すべての行列は対称
④ 固有値は必ず実数
ANSWER ② Eckart-Young 定理（1936）：任意の行列 A に対し、Frobenius ノルム（または2-ノルム）で最良の rank-k 近似は、SVD の上位 k 特異値を残し残りを 0 にしたもの。これが画像圧縮、レコメンドシステム、LoRA の理論的根拠。
PROBLEM 18
3層 NN の順伝播 h = ReLU(Wx + b) における W が 64×128, x ∈ ℝ128, b ∈ ℝ64 のとき、出力 h の次元は。
① 128　 ② 64　 ③ 192　 ④ 8192
ANSWER ② 行列積 Wx は (64×128)×(128) = ℝ64。これに b ∈ ℝ64 を足し、要素ごとの ReLU を通すため、出力次元は 64。
PROBLEM 19
マハラノビス距離 d_M(x, μ) = √((x-μ)TΣ⁻¹(x-μ)) において Σ = I のとき、d_M は何に一致するか。
① マンハッタン距離　 ② ユークリッド距離　 ③ チェビシェフ距離　 ④ 0
ANSWER ② Σ = I のとき (x-μ)TI⁻¹(x-μ) = ||x-μ||² で、その平方根は通常のユークリッド距離。マハラノビス距離は『データの共分散構造を考慮した距離』であり、共分散が単位行列なら全方向同等で、距離もユークリッドに一致する。
PROBLEM 20
『高次元での距離の呪い』の本質的な問題は。
① 計算速度が遅い
② 高次元では点同士の距離が均一化し、最近傍探索などの距離ベース手法が機能しなくなる
③ メモリが不足する
④ ファイルが大きい
ANSWER ② 高次元では距離の最大/最小の比が 1 に近づき、すべての点がほぼ等距離になる。これにより k-NN、K-means などが意味を持たなくなる。対処は次元削減（PCA, t-SNE, UMAP, Autoencoder）、特徴選択、距離計量学習（Metric Learning）。

章のまとめ

本章では、機械学習を支える線形代数の中核概念を10節で扱いました。スカラー・ベクトル・行列・テンソルの階層 から始まり、内積（類似度・Attentionの核）、行列積（NNの順伝播）、ノルム（正則化の幾何）、線形独立と階数（情報の重複）、行列式と逆行列（写像の体積）、固有値分解（PCA・スペクトラル解析）、SVD（万能の分解）、二次形式と正定値性（凸最適化の判定）。応用では NN、PCA、共分散、Self-Attention の Q·K·V構造を扱い、ハンズオンで NumPy・固有値分解・SVD画像圧縮・PCAスクラッチを実装、シミュレーションで距離の呪い・ランク削減・テンソル形状を体感しました。次章では微分・最適化、その後の章で確率・情報理論・古典機械学習・NN・畳み込み・Transformer へと進みます。すべての基礎は、ここで扱った行列の言葉です。
