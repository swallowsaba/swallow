章7 — 機械学習のパラダイムChapter 7 — ML Paradigms

機械学習には根本的に異なる 3つの学習パラダイム（教師あり・教師なし・強化学習）と、近年台頭した 自己教師あり学習があります。それぞれが何を入力し、何を学び、何に使われるのかを地図として持つと、現代AIの全景が一気に見えるようになります。

§1 / MAP機械学習の地図

FIG 7.1 機械学習の4大パラダイム。教師あり・教師なし・強化学習が古典的3分類、自己教師あり学習が近年加わった第4の柱（BERT, SimCLR, MAE などの基礎）。

§2 / SUPER教師あり学習

入力 x と正解ラベル y のペア (x, y) から、関数 f: x → y を学ぶ。最も一般的なパラダイム。

  • 分類：y がカテゴリ。「この画像は猫」「このメールはスパム」
  • 回帰：y が連続値。「明日の気温」「不動産価格」

代表的手法（章8で詳述）：線形/ロジスティック回帰、SVM、決定木、ランダムフォレスト、勾配ブースティング、ニューラルネット。

§3 / UNSUP教師なし学習

ラベルなしの x だけ から、データの構造を見つける。

  • クラスタリング：K-means、階層クラスタ、DBSCAN — 似たデータをグループ化
  • 次元削減：PCA、t-SNE、UMAP — 高次元データを低次元で可視化
  • 密度推定：分布を推定。GMM、KDE
  • 生成モデル：GAN、VAE、Diffusion — 新しいサンプル生成

§4 / RL強化学習

エージェントが環境とやり取りし、報酬を最大化する方策（policy） を学ぶ。教師ラベルではなく、遅延された報酬シグナル から学ぶのが特徴。

FIG 7.2 強化学習の基本構造。エージェントが行動 a を選び、環境が次状態 s と報酬 r を返す。累積報酬を最大化する方策を学ぶ。AlphaGo, ChatGPT の RLHF, ロボティクスの基盤。

4.1 主な手法

  • 価値ベース：Q学習、DQN — 各状態-行動の価値 Q(s,a) を推定
  • 方策ベース：REINFORCE、PPO — 方策 π を直接最適化
  • Actor-Critic：A2C、SAC — 価値と方策の両方
  • モデルベース：Dyna、World Models — 環境のモデルを学習して計画

4.2 主な応用

  • ゲーム：AlphaGo（囲碁）、AlphaZero、MuZero、AlphaStar（StarCraft）
  • RLHF：ChatGPT, Claude の人間フィードバック学習
  • ロボット制御：歩行、操作、自動運転
  • 推薦システム：長期的なユーザー満足度の最大化

§5 / SELF自己教師あり学習

「人手でラベルを付けない」が「データ自身から擬似ラベルを生成して教師あり学習する」枠組み。近年の事前学習（BERT、GPT、SimCLR、MAE、DINO）はほぼ全てこれ。

タスク擬似ラベルの作り方例
Masked Language Model文の一部の単語を [MASK] にして、それを予測BERT, RoBERTa
Next Token Prediction次の単語を予測GPT 系全て
Masked Image Model画像のパッチを隠して復元MAE, BEiT
対比学習同じ画像の2つの増強を近づけ、別画像を遠ざけるSimCLR, MoCo
クロスモーダル画像とキャプションのペアを近づけるCLIP, ALIGN

なぜ重要か：人手アノテーションは高コストで質に限界がある。インターネットには ラベルなしのテキスト・画像 が無限にある。それを使えるなら学習データを桁違いに増やせる。これが 基盤モデル（FM）の革命 の正体。

§6 / SEMI半教師あり学習

少量のラベル付きデータと大量のラベルなしデータを併用。医療画像など アノテーションが高価な分野 で重要。

代表手法：

  • 擬似ラベリング（モデルの予測を擬似ラベルに）
  • 整合性正則化（ノイズに対する予測の安定性）
  • FixMatch、Mean Teacher

§7 / ERM経験リスク最小化

教師あり学習の数学的定式化：

  • 仮説空間 H：考えるモデル族（線形回帰、決定木、NN など）
  • 損失関数 L(f(x), y)：予測誤差の定義
  • 経験リスク：訓練データ上の平均損失
f̂ = argmin_{f ∈ H} (1/n) Σ L(f(xᵢ), yᵢ)

しかし真に最小化したいのは 汎化誤差（未知データでの誤差）。過学習 はこのギャップ（章9で扱う）。

§8 / SUMMARY章のまとめ

1機械学習の4パラダイム：教師あり／教師なし／強化学習／自己教師あり
2教師あり：(x, y) ペアから学ぶ。分類と回帰
3教師なし：x のみ。クラスタ・次元削減・生成
4強化学習：エージェント・環境・報酬のループ
5自己教師あり：データ自身から擬似ラベル。BERT, GPT, MAE
6半教師あり：少量ラベル + 大量ラベルなし
7経験リスク最小化（ERM）が教師あり学習の定式化
8RLHF が ChatGPT などのアライメントの中核

  1. 教師あり学習・教師なし学習・強化学習の入力と出力を比較できるか
  2. 「クラスタリング」は教師ありか教師なしか
  3. 強化学習の3要素（エージェント、環境、報酬）を図で説明できるか
  4. BERT のマスク言語モデルが「自己教師あり」と呼ばれる理由
  5. CLIP がどんなタスクで学習しているか（クロスモーダル対比）
  6. 半教師あり学習が有効な実務シーンの例
  7. 経験リスク最小化の式を書けるか
  8. RLHF が「強化学習」の枠に入る理由
