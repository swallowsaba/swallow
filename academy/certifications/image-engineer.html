画像処理エンジニア検定Image Processing Engineer (CG-ARTS)

CG-ARTS 協会が主催する画像処理の知識検定。ベーシック／エキスパートの 2 階級で、サンプリング・フィルタ・エッジ・形状解析といった古典的画像処理の体系と、近年は深層学習による認識までを横断的に問います。CV エンジニアの基礎教養として最適。

§02 / BASICベーシック

画像処理の基礎概念と古典的アルゴリズムが中心。学生・初学者・隣接分野エンジニアの入門に最適。

  • 画像表現（標本化・量子化、色空間、ファイルフォーマット）
  • 濃淡変換（ヒストグラム、ガンマ補正）
  • 空間フィルタ（平均、ガウシアン、メディアン、ラプラシアン）
  • 二値化、モルフォロジー、ラベリング
  • 幾何変換（平行移動・回転・拡大縮小、アフィン、補間）
  • 圧縮の概念（JPEG/PNG/GIFの違い）

§03 / EXPERTエキスパート

ベーシックを土台に、より高度な処理と応用、近年は深層学習までを問う。

  • 周波数領域処理（FFT、DCT、ウェーブレット）
  • エッジ検出（Sobel、Canny、LoG、Hough 変換）
  • 特徴記述（SIFT、SURF、ORB、HOG、LBP）
  • セグメンテーション（領域分割、グラフカット、SLIC）
  • 動画像処理（オプティカルフロー、MPEG、背景差分）
  • 3次元復元（ステレオ、SfM、Structure from Motion）
  • 機械学習・深層学習による画像認識（CNN、物体検出、セグメンテーション、生成）

§04 / FUND基礎概念

標本化と量子化

  • ナイキスト定理: f_s ≥ 2 f_max。これを下回るとエイリアシング
  • 量子化ビット深度（8/10/12/16bit）→ ダイナミックレンジ

輝度の表現

  • ガンマ補正: 表示と人間視覚の非線形性を補正（sRGB は概ね γ≈2.2）
  • ヒストグラム平坦化、CLAHE で局所コントラスト改善

解像度とファイルサイズ

  • 非圧縮データ量 = 幅 × 高さ × チャネル × ビット深度
  • JPEG は非可逆（DCT＋量子化）、PNG は可逆（DEFLATE）、GIF は限定色＋アニメ

§05 / FILTERフィルタとエッジ

平滑化

  • 平均（ボックス）: 単純だが境界もぼやける
  • ガウシアン: σ で広がり制御、分離可能で高速
  • メディアン: 塩胡椒ノイズに頑健
  • バイラテラル / Non-local Means: エッジ保持平滑化

エッジ検出

  • Sobel/Prewitt/Scharr: 一次微分
  • Laplacian / LoG: 二次微分のゼロ交差
  • Canny: 平滑化→勾配→非極大抑制→二重閾値・ヒステリシス
  • Hough 変換: パラメータ空間投票で直線・円検出

§06 / FEATURE特徴抽出とマッチング

  • SIFT: スケール・回転・照明に頑健、DoG ピラミッド＋勾配ヒストグラム
  • SURF: SIFT 高速版（積分画像）
  • ORB: 特許フリー、FAST + BRIEF、二値特徴
  • HOG: 勾配方向ヒストグラム、歩行者検出（Dalal-Triggs）
  • LBP: 局所2値パターン、テクスチャに強い
  • RANSAC: 外れ値を含むデータでのモデル推定（ホモグラフィ等）

§07 / SHAPE形状と幾何変換

形状解析

  • 二値化（大津、適応的、Sauvola、Niblack）
  • モルフォロジー（dilation/erosion、opening/closing、top-hat）
  • 連結成分ラベリング（4近傍／8近傍、Union-Find）
  • 輪郭追跡、面積・周囲長・形状特徴（Hu モーメント）

幾何変換

  • アフィン（6自由度、平行性保存）
  • 射影（ホモグラフィ、8自由度）
  • 補間（最近傍、双線形、双三次、Lanczos）
  • イメージスティッチング: 特徴対応＋ホモグラフィ＋ブレンディング

§08 / COLORカラーとコーデック

  • 色空間: RGB / HSV / YCbCr / Lab、CIE 色度図
  • JPEG は YCbCr 変換＋4:2:0サブサンプリング＋DCT＋量子化＋エントロピー符号化
  • 動画コーデック: H.264 / H.265 (HEVC) / AV1、I/P/B フレーム、GOP、動き補償
  • HDR: PQ / HLG、広色域（BT.2020）

§09 / DLディープラーニング応用

  • CNN: 畳み込み・プーリング、ResNet、EfficientNet、ConvNeXt
  • 物体検出: Faster R-CNN（2段階）、YOLO・SSD・DETR（1段階）、NMS・IoU・mAP
  • セグメンテーション: FCN、U-Net、DeepLab、Mask R-CNN、Panoptic
  • Vision Transformer: ViT、Swin、MAE。事前学習データ規模が成功要因
  • マルチモーダル: CLIP、BLIP、LLaVA
  • 生成: GAN、VAE、Diffusion（Stable Diffusion）＋ControlNet
  • 3D / NeRF / Gaussian Splatting: 視点合成・3D 復元の最新潮流

§10 / ROADMAP学習ロードマップ

Phase 1 — ベーシック（40〜60時間）

『ディジタル画像処理（CG-ARTS）』を写経しながら通読。OpenCV でフィルタ・エッジ・二値化・モルフォロジーを手元で確認。

Phase 2 — エキスパート（80〜120時間）

同書のエキスパート章、Szeliski 『Computer Vision: Algorithms and Applications』の主要章。SIFT、Hough、Stereo、SfM を OpenCV で実装。

Phase 3 — 深層学習応用（60〜100時間）

PyTorch + torchvision で分類／検出／セグメンテーションを各 1 本実装。Hugging Face で ViT / DINOv2 / SAM を試す。
