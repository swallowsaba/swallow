§01 / OVERVIEW検定の概要

名称統計検定（日本統計学会公式認定）
運営一般財団法人 統計質保証推進協会
主な階級4級 / 3級 / 2級 / 準1級 / 1級 / 統計調査士 / 専門統計調査士 / データサイエンス基礎・発展・エキスパート
形式多くの級は CBT（随時受験、即時採点）。1級のみ年1回（例年6月）の筆記論述
受験資格制限なし（年齢・学歴問わず）
合格基準概ね 60点 / 100点（級・分野で微差あり）
本サイトの推奨AI志望は 2級 → 準1級 → DSエキスパート、研究志向なら 1級
統計を学ぶ意味

機械学習モデルが出した値の信頼区間を読めない、A/Bテストの p値を誤解する、回帰係数の符号を物理的に解釈できない――こうした失敗は、すべて「統計の側」を埋めることで防げます。統計検定はその訓練の標準化された場であり、合格証以上に 学習プロセス自体に価値 があります。

§02 / LEVELS各級の位置づけ

  • 3級 — 高校レベル。記述統計、確率の入口。
  • 2級 — 大学基礎レベル。確率分布、点推定・区間推定、仮説検定、回帰、ANOVA。データサイエンスの最低ライン。最も受験者の多い級。
  • 準1級 — 多変量解析、時系列、ベイズ、実験計画、ノンパラ、GLM。実務寄りで E資格 の数学領域と最も重なる。
  • 1級 — 大学院レベル。「統計数理」と「統計応用（人文／社会／医薬／理工 から1選択）」の 2 科目論述。最難関。導出を書ける力が必要。
  • DS基礎 / 発展 / エキスパート — Python・R での計算を伴う実務志向試験。エキスパートは大規模データのモデリング全般。
  • 統計調査士・専門統計調査士 — 調査設計・公的統計・サンプリング設計に特化。

§03 / LEVEL 22級の詳細

主要トピック

  • 記述統計（中心傾向・散らばり・歪度・尖度）／相関・共分散／単・重回帰の基本
  • 確率分布（離散：二項・ポアソン・幾何／連続：正規・指数・一様）
  • 標本分布（標本平均、t、χ²、F）／中心極限定理
  • 点推定（最尤・モーメント法）／区間推定（信頼区間の正しい解釈）
  • 仮説検定（1標本・2標本、対応あり／なし、t検定、Welch、F検定、比率の検定）
  • カテゴリカルデータ（独立性の検定、適合度検定）
  • 単・重回帰（決定係数、調整済み R²、F統計量、残差プロット）
  • 一元・二元分散分析（主効果・交互作用）

頻出のひっかけ・注意点

  • 「真の値が95%の確率で区間内」は 誤り。正しくは「同じ手順を多数回行うと95%の区間が真の値を含む」
  • p値は H0 の確率ではなく、H0 が真と仮定したときに観測値以上の結果が得られる確率
  • 独立性の検定では 期待度数が5以下のセル が多い場合 Fisher 正確検定を検討
  • 等分散性に疑義があれば Welch の t検定 を選択

§04 / LEVEL PRE-1準1級の詳細

2級の延長に「本格的なモデリングと多変量」が加わります。AI/ML との接続が最も濃い級。

追加で問われるテーマ

  • 多変量解析: 主成分分析（PCA）、因子分析、判別分析（LDA/QDA）、クラスター分析（階層・K-means）、正準相関
  • ノンパラメトリック検定: 符号検定、Wilcoxon、Mann-Whitney U、Kruskal-Wallis、ブートストラップ
  • 時系列解析: 自己相関・偏自己相関、AR/MA/ARMA/ARIMA、定常性と単位根（ADF）、状態空間モデル
  • 実験計画: 完全/不完全ブロック、乱塊法、ラテン方格、直交配列、応答曲面法、SN比
  • ベイズ統計: 事前事後分布、共役分布、最尤と MAP、MCMC（Metropolis-Hastings、Gibbs）の入門
  • 一般化線形モデル（GLM）: ロジスティック、ポアソン、負の二項、リンク関数、逸脱度
  • モデル選択・検証: AIC、BIC、Cp、交差検証、ブートストラップ
  • 因果推論: 傾向スコア、IPTW、差の差、操作変数（基本まで）
準1級と E資格の相補性

準1級は「統計の側から」、E資格は「最適化と DL の側から」、同じ数学的中身に到達します。両方を学ぶことで、論文の理解、データの誠実な扱い、安全なモデル運用の能力が立体的に整います。

§05 / LEVEL 11級の詳細（論述）

例年 6 月開催。統計数理と 統計応用（人文／社会／医薬／理工 から1選択）の 2 科目。論述形式で、導出を書ける力 が問われます。

統計数理

  • 確率論：測度論の入口、確率変数、確率収束、極限定理
  • 推定論：MLE、十分統計量、Fisher 情報量、Cramér-Rao 下限、漸近正規性
  • 仮説検定：Neyman-Pearson の基本定理、検出力、UMP 検定、尤度比検定
  • ベイズ：事前選択、階層モデル、ベイズ的検定
  • 線形モデルの行列表現、最小二乗の幾何

統計応用（選択）

  • 人文科学: 心理測定、因子分析、構造方程式モデル
  • 社会科学: 標本調査、計量経済（操作変数、時系列）
  • 医薬生物: 生存解析、ロジスティック、臨床試験のデザイン
  • 理工学: 信頼性、品質工学、実験計画、応答曲面

§07 / KEY頻出概念リファレンス

記述統計

  • 平均 μ = Σxᵢ/n、分散 σ² = Σ(xᵢ-μ)²/n、不偏分散は分母を n-1
  • 標準化 z = (x - μ) / σ で平均0・分散1
  • 四分位範囲 IQR = Q3 - Q1。外れ値の目安 1.5 × IQR
  • ピアソン相関は線形のみを捉える。非線形ならスピアマンや相互情報量

分布

  • 二項 Bin(n,p)：平均 np、分散 np(1-p)
  • ポアソン Po(λ)：平均=分散=λ。希少事象の到着
  • 正規 N(μ,σ²)：左右対称、平均=中央値=最頻値。CLTで標本平均が漸近正規
  • t分布：自由度小で裾が重く、∞で正規

推定と検定

  • 標準誤差 = σ/√n。検出力 = 1-β
  • Bonferroni 補正は FWER 制御、Benjamini-Hochberg は FDR 制御
  • 母分散既知 → z検定、未知 → t検定

回帰・GLM

  • OLS は残差二乗和の最小化、正規ノイズ仮定で MLE と一致
  • 多重共線性は VIF≥10 が目安、リッジで縮小、ラッソでスパース化
  • ロジスティック回帰の出力は P(Y=1|X)、ポアソン回帰のリンクは log

ベイズ

  • 事後 ∝ 尤度 × 事前。MAP は事前が一様なら MLE に一致
  • 共役: ベルヌーイ↔ベータ、正規↔正規、ポアソン↔ガンマ
  • MCMC は近似サンプリング、VI は ELBO 最大化による変分近似

時系列

  • 弱定常 = 平均・分散・自己共分散が時間によらない
  • ADF の H0 は「単位根がある」＝非定常
  • ACF・PACF で AR/MA の次数推定（Box-Jenkins）

因果推論

  • RCT は介入のランダム化が因果の根拠
  • 傾向スコア（マッチング・IPTW）、差の差（並行トレンド前提）、IV（内生性回避）、RDD（カットオフ近傍）

§08 / ROADMAP学習ロードマップ

Phase 1 — 2級到達（40〜80時間）

『統計学入門（東京大学出版会・基礎統計学Ⅰ）』を通読、確率分布の積分・微分が手で動かせる 状態に。公式問題集を 2 周。CBT 模試で時間配分を体得。

Phase 2 — 準1級到達（80〜120時間）

『統計学実践ワークブック』を主軸に、Python / R で各手法を実装。ロジスティック・ポアソン・ARIMA・PCA・LDA・MCMC を一通り実装することで理解が深まる。E資格の数学パートと同時並行が効率的。

Phase 3 — 1級挑戦（200時間〜）

『現代数理統計学の基礎』(久保川) などで導出に慣れる。過去問の論述を 手で書き起こす 訓練。応用分野（医薬／理工等）の専門書 1 冊を読み込む。

Phase 4 — DSエキスパート挑戦（合算150〜250時間）

Python / R での実装力＋因果推論＋深層学習評価。Kaggle や実データ分析プロジェクトを並走させると盤石。
